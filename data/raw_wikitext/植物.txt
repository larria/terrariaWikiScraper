== 生长 ==
所有类型的{{tr|plants}}、[[{{tr|seashells}}]]、[[{{tr|Coral}}]]的生长、和其他随时间生成的{{tr|tiles}}都取决于每{{tr|tick}}的随机{{tr|tile}}更新。{{tick}}{{source code ref|v=1.4.4.9|method=Terraria.WorldGen.UpdateWorld}}总体来说，每{{tr|tick}}更新更多{{tr|tile}}意味着更快的生长。

{{tr|The tile update}}分为两个部分：
* {{anchor|overground}}“地面上”{{tr|updates}}{{source code ref|v=1.4.4.9|method=Terraria.WorldGen.UpdateWorld_OvergroundTile}}，这包括通常发生于[[{{tr|surface}}]]{{tr|layer}}及以上的{{tr|updates}}。
* {{anchor|subterranean}}“地面下”{{tr|updates}}{{source code ref|v=1.4.4.9|method=Terraria.WorldGen.UpdateWorld_UndergroundTile}}，这包括通常发生于[[{{tr|underground}}]]{{tr|layer}}及以下的{{tr|updates}}。

<code>更新数 = {{tr|World}}宽度 × {{tr|World}}高度 × 更新速率 × {{tr|Journey Mode}}时间速率</code>

== {{tr|Moss}} ==
[[{{tr|Moss}}]]是生长在[[{{tr|Stone Block}}]]上的类{{tr|grass}}{{tr|plant}}。它有多种不同的颜色，会随机地出现在{{tr|cave surfaces}}上或有相应{{tr|mossy wall}}的[[{{tr|moss chamber}}]]中{{tr|Moss}}会长出毛绒绒的植被，可以像割杂草那般割掉。在{{eversions|1.4.0.1}}中，可以用[[{{tr|Paint Scraper}}]]来将其作为{{tr|item}}收集，并用于重新种植或制作{{tr|mossy walls}}。

== {{tr|Herbs}} ==
[[{{tr|Herbs}}]]用于在[[{{tr|Placed Bottle}}]]或[[{{tr|Alchemy Table}}]]处制作[[{{tr|potions}}]]。他们出现在世界各地，或出现在{{eil|Herb Bag|s}}中，可以用几乎任何[[{{tr|weapon}}]]或[[{{tr|tool}}]]收割。每种{{tr|herb}}都会在符合某些要求时开花。在开花时收割{{tr|Herbs}}会额外掉落 1–3 颗相应[[{{tr|seeds}}]]。

== {{tr|Mushrooms}} ==
[[{{tr|Mushrooms|link=y}}|{{tr|Mushrooms}}]]是随机生长在很多[[{{tr|grasses}}]]和其他[[{{tr|blocks}}]]上的{{tr|plants}}。[[{{tr|Biome spread}}]]或人工更改{{tr|biome}}的方法（[[{{tr|solutions}}]]、[[{{tr|Thrown Powder}}]]、[[{{tr|Thrown Water}}]]，等等）能够使常规[[{{tr|Mushroom}}]]、[[{{tr|Vile Mushroom}}]]、和[[{{tr|Vicious Mushroom}}]]互相转化，并使[[{{tr|Glowing Mushroom}}]]和[[{{tr|Jungle Spores}}]]互相转化。

共有 6 种{{tr|mushrooms}}：

'''{{tr|Vines}}'''是自{{tr|blocks}}向下方生长的植物。如果有足够的空间，它们最多可以长到十格长。尽管它们会阻碍光线（会发光的{{tr|Glowing Mushroom vines}}除外），但他们不会妨碍移动并且可以用几乎任意[[{{tr|weapon}}]]或[[{{tr|tool}}]]轻松摧毁。摧毁一格{{tr|vines}}会摧毁其下方的所有{{tr|vines}}，摧毁最上方的一格或者挖掉它所悬挂的{{tr|block}}会摧毁整株{{tr|vines}}。

总共有{{eicons|1.4.4}}八 / {{eicons|1.4.4|invert=y}}四种不同类型的{{tr|vines}}：{{tr|Regular vines}}（从[[{{tr|Grass}}]]和[[{{tr|Leaf Block}}]]上长出）、{{tr|Jungle vines}}（从[[{{tr|Jungle grass}}]]块上长出）、{{tr|Hallowed vines}}（从[[{{tr|Hallowed grass}}]]块上长出）、{{tr|Crimson vines}}（从[[{{tr|Crimson grass}}]]块上长出）、{{eil|nl=y|Flower vine|s}}（从长草洞穴中的[[{{tr|Grasses}}]]上或玩家放置的[[{{tr|Grass Wall}}]]或[[{{tr|Flower Wall}}]]前长出）、{{eil|nl=y|Mushroom vine|Glowing Mushroom vines}}（从[[{{tr|Mushroom grass}}]]上长出）、{{eil|nl=y|Corrupt vine|s}}（从[[{{tr|Corrupt grass}}]]上长出）、和{{eil|nl=y|Ash vine|s}}（从[[{{tr|Ash grass}}]]上长出)。

{{tr|Vines}}只能在[[{{tr|surface}}]]以上生长（{{tr|Jungle vines, Glowing Mushroom vines, and Ash vines}}例外，它们可以在任何高度生长）并且只能从下方不是[[{{tr|Hammers}}|斜面]]的{{tr|blocks}}上长出。

{{tr|Vines}}可以被[[{{tr|paint}}|涂漆]]。从涂漆的{{tr|blocks}}中新长出来的{{tr|vines}}会具有该{{tr|block}}的颜色而{{tr|vines}}自然延伸所生成的{{tr|vines}}图格将具有{{tr|vines}}现有尖端的颜色。

{{tr|Jungle vines tiles}}计入其[[生物群落的存在要求]]，但{{tr|Corrupt vines, Crimson vines, and Hallowed vines}}不计。{{source code ref|v=1.4.4.5|method=Terraria.SceneMetrics.ExportTileCountsToMain}}不过，这些{{tr|vines}}会向附近的{{tr|block}}[[{{tr|biome spread}}|{{tr|spread}}]]其对应的{{tr|biome}}。{{eil|Guide to Plant Fiber Cordage}}允许从{{tr|regular vines and Jungle vines}}中收集{{eil|Vine Rope|s}}。

{{tr|Vines}}永远不会掉落[[{{tr|Vine}}]]物品，后者是[[{{tr|Man Eater}}]]的不常见掉落物。不过，{{tr|Glowing Mushroom vines}}在被摧毁时会掉落[[{{tr|Glowing Mushroom}}]]。

== {{tr|Cattails}} ==
{{exclusive|1.4.0.1|section=y}}
[[File:Catttails in a lake.png|thumb|湖中的{{tr|Forest cattails}}。]]

'''{{tr|Cattails}}'''是在淹没于[[{{tr|water}}]]下 1–7 格深的{{tr|block}}上自然生长出来的{{tr|plants}}。它们是{{tr|background objects}}，可以用绝大多数[[{{tr|weapons}}]]或[[{{tr|tools}}]]摧毁。摧毁{{tr|cattail}}的某一格会同时摧毁{{tr|cattail}}在这一格以上的所有{{tr|tile}}，而摧毁其底部或者其所附着的{{tr|block}}会完全摧毁此{{tr|cattail}}。

有六种不同类型的{{tr|cattails}}：{{tr|Regular cattails}}（从[[{{tr|Grasses}}|{{tr|Grass}}]]上长出）、{{tr|Jungle cattails}}（从[[{{tr|Jungle grass}}]]和[[{{tr|Oasis}}]]中的[[{{tr|Sand Blocks}}]]上长出）、{{tr|Hallowed cattails}}（从[[{{tr|Hallowed grass}}]]上长出）、{{tr|Glowing Mushroom cattails}}（从[[{{tr|Mushroom grass}}]]上长出）、{{tr|Corruption cattails}}（从[[{{tr|Corrupt grass}}]]上长出）和{{tr|Crimson cattails}}（从[[{{tr|Crimson grass}}]]上长出）。除了{{tr|Glowing Mushroom cattails}}外，所有{{tr|cattails}}都只能在{{tr|surface}}以上生长。

{{tr|Glowing Mushroom cattails}}被摧毁时会掉落[[{{tr|Glowing Mushroom}}]]；其他类型的{{tr|cattails}}不产生任何掉落。

== {{tr|Lily pads}} ==
{{exclusive|1.4.0.1|section=y}}
[[File:All lilypad variants.png|thumb|所有类型{{tr|Lily pad}}的变种。从上至下：常规、{{tr|Hallowed, Jungle, Crimson, and Corruption lily pads}}]]

'''{{tr|Lily pads}}'''是在水面以下 3–12 格内有合适{{tr|block}}时生长在水面的{{tr|plant}}，它们是可以被大多数[[{{tr|weapon}}]]和[[{{tr|tool}}]]破坏的{{tr|background object}}。

总共有五种不同类型的{{tr|Lily pads}}，每种{{tr|Lily pads}}有 18 种变种，共 90 种。它们是： {{tr|Regular lily pads}}（在[[{{tr|Grass}}]]上方）、{{tr|Jungle lily pads}}（在[[{{tr|Jungle grass}}]]和[[{{tr|Oasis}}]]中的[[{{tr|Sand Blocks}}]]上方）、{{tr|Hallowed lily pads}}（在[[{{tr|Hallowed grass}}]]上方）、{{tr|Corruption lily pads}}（在[[{{tr|Corrupt grass}}]]上方）和{{tr|Crimson lily pads}}（在[[{{tr|Crimson grass}}]]上方）。

'''{{tr|Thorny bushes}}'''，或'''{{tr|thorns}}'''是会伤害任何接触它的玩家的{{tr|plants}}。共有{{eicons|1.4.4}}四/{{eicons|1.3.0.1|invert=y}}两种不同的{{tr|thorny bushes}}：
* [[{{tr|Underground Jungle}}]]中的棕色灌木造成 17 {{tr|base damage}}。
* [[{{tr|the Corruption}}]]中的紫色灌木造成 10 {{tr|base damage}}。
* {{eicons|1.3.0.1}} [[{{tr|the Crimson}}]]中的红色灌木造成 10 {{tr|base damage}}。
* {{eicons|1.4.4}} 来自 [[For the worthy]] {{tr|world}}中的[[{{tr|Plantera}}]]的粉色灌木造成 100 {{tr|base damage}}。{{source code ref|v=1.4.4.6|method=Terraria.Player.ApplyTouchDamage|field=Terraria.ID.TileID.Sets.TouchDamageImmediate}}

{{tr|Thorny bushes}}会在被[[{{tr|weapon}}]]、[[{{tr|tool}}]]、或玩家接触后被摧毁，但隐形的{{tr|projectiles}}（如[[{{tr|Medusa Head}}]]、[[{{tr|Life Drain}}]]等）不会摧毁它们。它们也无法作为物品获得。所有 [[{{tr|NPCs}}]]、[[{{tr|enemies}}]]、和[[{{tr|critters}}]]都不会受到{{tr|thorny bushes}}的伤害，也无法将其摧毁。特别地，所有{{tr|Plantera}}的{{tr|thorny bushes}}会在{{tr|Plantera}}不存在时被摧毁。{{source code ref|v=1.4.4.9|method=Terraria.WorldGen.UpdateWorld_OvergroundTile / Terraria.WorldGen.UpdateWorld_UndergroundTile}}{{tr|thorny bushes}}可以在任意方向上长出六格远，但其生长会被[[实心物体]]所阻挡。和[[{{tr|wild grass plants}}]]不同，在{{tr|soil block}}后方有{{tr|background walls}}时，{{tr|thorny bushes}}仍然能生长。

紫色和红色的种类会[[{{tr|Biome spread}}|蔓延{{tr|Corruption}}/{{tr|Crimson}}]]；但是棕色的种类不会蔓延[[{{tr|Jungle grass}}]]。

[[{{tr|Strange Plant}}]]是不常见的[[{{tr|Hardmode}}]]{{tr|quest items}}，可以用于与[[{{tr|Dye Trader}}]]交易以换取 33 种特殊[[{{tr|dyes}}]]中的 1 种，获得的{{tr|dye}}种类与已击败的 [[{{tr|bosses}}]] 和/或[[{{tr|events}}]]相关。{{tr|player}}会获得 6 个同种随机选择的{{tr|dye}}。

== {{tr|Plantera's Bulb}} ==
{{float file box|width=8em|[[File:Plantera's Bulb.gif]]|{{tr|Plantera's Bulb}}}}

[[{{tr|Plantera's Bulb}}]]是[[{{tr|Underground Jungle}}]]中的一种粉色{{tr|plant}}，它会在当前世界处于[[{{tr|Hardmode}}]]下，且已经击败所有 3 个[[{{tr|mechanical bosses}}]]后（{{3DS version}}中只需击败 1 个）出现。它在被{{tr|pickaxe}}/{{tr|drill}}采掘时、被{{tr|explosives}}破坏时、被[[{{tr|Clentaminator}}]]喷射到[[{{tr|Solution}}]]时，或其下方的{{tr|Jungle grass}}被锤击或转换为另一个{{tr|block}}时会召唤[[{{tr|Plantera}}]]。虽然它不能被{{tr|weapons}}或{{tr|projectiles}}破坏，但仍需小心对待，以避免不必要的 {{tr|boss}} 战。大约每六分钟会生成一个{{tr|Plantera's Bulb}}（约为游戏内的四分之一天）。

{{-}}

== {{tr|Pumpkin}} ==
{{float file box|[[File:Pumpkin Growth.gif]]|{{tr|pumpkin}}生长的各个阶段。}}

[[{{tr|Pumpkins}}]]在{{tr|Halloween}}期间会自然生成，并且可以用[[{{tr|Dryad}}]]所售的[[{{tr|Pumpkin Seed}}]]来人工种植。收获{{tr|Pumpkins}}会掉落{{tr|Pumpkin block}}，它可以被放置，也可以制作成[[{{tr|Pumpkin armor}}|{{tr|armor}}]]、[[{{tr|Pumpkin furniture}}|{{tr|furniture}}]]、[[{{tr|Pumpkin Wall}}|{{tr|walls}}]]、[[{{tr|Pumpkin Pie}}]]、和[[{{tr|Pumpkin Moon Medallion}}]]。收获{{tr|Pumpkins}}还有几率掉落[[{{tr|Magical Pumpkin Seed}}]]。
