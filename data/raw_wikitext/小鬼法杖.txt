'''{{tr|Imp Staff}}'''是一个[[{{tr|pre-Hardmode}}]]的[[{{tr|summon weapon}}]]，可召唤'''{{tr|Flying Imp}}'''[[{{tr|minion}}]]。它徘徊在[[{{tr|enemy}}]]周围，发射{{tr|fireballs}}，可施加 3-6 {{tr|seconds}}的[[{{tr|On Fire!}}]][[{{tr|debuff}}]]，并可穿透 4 个{{tr|enemies}}。

其最佳[[{{tr|modifiers}}]]是'''{{tr|Mythical}}'''， '''{{tr|Furious}}'''，'''{{tr|Masterful}}'''或'''{{tr|Godly}}'''。取整使得这些{{tr|modifiers}}提供和{{tr|Ruthless modifier}}相同的伤害增幅，将伤害从 17 增至 20。

== 备注 ==
* {{tr|The Imp}}平均每 {{duration|67.66 ticks}}攻击一次。每两次攻击间的延迟在 {{duration|45 ticks}}-{{duration|90 ticks}}之间。{{source code ref|v=1.4.4.9|method=Terraria.Projectile.AI_062|t={{tr|imp}}的 AI[1] 每{{tr|tick}}加 1 并且有 1/3 的概率在同一{{tr|tick}}内再加 1 。所以其每{{tr|tick}}平均加 4/3 。其阈值为 90，所以平均每两次攻击间的冷却为 67.5 {{tr|ticks}}。}}
* {{tr|Imp Fireball}}击中{{tr|enemy}}时会造成 6 {{tr|tick}}{{tick}}[[{{tr|Static invincibility frames}}]]。由于[[{{tr|Damage}}#“穿透”敌怪的攻击|穿透攻击]]的运作机制，多个{{tr|imps}}的攻击会互相干扰。
* 虽然{{tr|Imp}}能够在[[{{tr|water}}]]中飞行，其{{tr|fireball projectiles}}不能在{{tr|water}}行进。不过，{{tr|Imp}}仍然会尝试攻击水下的{{tr|enemies}}。
* {{tr|The Imp minions}}最远可以锁定距离{{tr|player}} 125 格的{{tr|enemy}}。{{source code ref|name=aistyle66|v=1.4.4.9|method=Terraria.Projectile.AI_062}}
** 对于{{tr|player}}标记的{{tr|enemy}}，此距离会增加至 187.5 格。
* 若{{tr|Imp minions}}距离{{tr|player}}超过 62.5 格，它们会先返回{{tr|player}}身侧再继续{{tr|attack}}。<ref name="aistyle66"/>
* {{tr|The Imp Fireballs}}移动速度为 {{mph|22 pixels}},且如果没有击中{{tr|enemy}}或{{tr|block}}会在 {{duration|150 ticks}}后消失。{{source code ref|v=1.4.4.9|method=Terraria.Projectile.AI_062 / Terraria.Projectile.SetDefaults}}
