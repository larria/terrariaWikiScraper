This page is used for displaying the dynamic tables of modifiers and their effects on the base page, [[Modifiers]].

It takes two unnamed parameters: a comma-separated list of stat names (see {{tl|modifierinfo}}) and a comma-separated list of modifier names.

When only passing one modifier name, the <code>$row</code> parameter can be used to prevent the subpage from wrapping its output in a separate table row. This is useful for usage within a table row.
