'''{{tr|Lunar Flare}}'''是一本[[{{tr|Hardmode}}]]，[[{{tr|Moon Lord}}]]后的[[{{tr|spell book}}]]，其有 {{chance|2/9}} 的概率掉落自[[{{tr|Moon Lord}}]]。使用时，其会召唤三枚{{tr|lunar flares projectile}}向{{tr|cursor}}位置倾泄而下。{{tr|The lunar flares}}与[[{{tr|enemies}}]]和[[{{tr|blocks}}]]碰撞时爆炸，但直到它们到达{{tr|cursor}}位置时可穿透{{tr|blocks}}，然后才会爆炸。

== 备注 ==
* {{tr|The Lunar Flare}}的{{tr|projectiles}}会发出大量的光，但穿过{{tr|solid blocks}}时不会发光。
* {{tr|The Lunar Flare}}的{{tr|projectiles}}击中时造成 -1 {{tr|ticks}}{{tick}}[[{{tr|local invincibility frame}}]]，这意味着每个{{tr|projectile}}不能多次伤害同一目标。{{source code ref|v=1.4.4.9|method=Terraria.Projectile.Damage}}
* {{tr|The Lunar Flare}}的爆炸可以伤害碰撞点水平和竖直方向 4.375 {{tr|tiles}}内的{{tr|enemy}}。{{source code ref|v=1.4.4.9|method=Terraria.Projectile.AI_001|t=爆炸覆盖以碰撞点为中心 8.75×8.75 {{tr|tiles}}的方形区域。}}

== 花絮 ==
* 它的名字（Lunar Flare，直译为月亮耀斑）模仿了现实中的天文现象{{wikipedia|lang=en|solar flare|太阳耀斑（solar flare）}}。
