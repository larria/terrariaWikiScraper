'''{{tr|Gravity}}'''对玩家施加一个恒定向下的加速度，影响垂直速度和达到最大下落速度的时间。默认的{{tr|Gravity}}加速度是 {{mph|90 tiles}} 每秒{{source code ref|v=1.4.4.9|method=Terraria.Player.defaultGravity}}，能达到 {{mph|37.5 tiles}}{{source code ref|v=1.4.4.9|method=Terraria.Player.update}} 的最大下落速度<!-- gravity = 0.4 tile/tick^2; maxFallSpeed = 10 tile/tick -->。某些因素可能会同时修正{{tr|Gravity}}和最大下落速度，导致增加或减少它们的影响效果。

=== {{tr|Space}} ===
计算{{tr|gravity}}倍率的公式为：<code>倍率 = (x - 偏移 - 60) * 因数1/因数2</code>。因此，最终结果会位于 0.25 至 1 之间（在 [[{{tr|Don't dig up}}]] 和 [[{{tr|Get fixed boi}}]] {{tr|worlds}}中为 0.1 至 1）。下方为参数的含义：{{source code ref|name=update|v=1.4.4.9|method=Terraria.Player.Update}}
* <code>x</code> 等于玩家与{{tr|world}}顶部边界的距离（单位为{{tr|tile}}）。
* <code>偏移</code> 由[[{{tr|world size}}]]决定，{{tr|large world}}为 20，{{tr|medium world}}约为 15，{{tr|small world}}为 10。
* <code>因数1</code> 指世界是否是 [[{{tr|Don't dig up}}]] 或 [[{{tr|Get fixed boi}}]]。普通{{tr|world}}为 6，上述{{tr|world}}为 1。
* <code>因数2</code> 等于{{tr|world}}[[{{tr|Surface}}]]（0 英尺）至{{tr|world}}顶部边界的距离。

通过使用[[{{tr|Gravitation Potion}}]]或装备{{eil|Gravity Globe}}，玩家会获得逆转{{tr|gravity}}的能力。按下 {{key|up}} 键会逆转玩家的{{tr|gravity}}，玩家会向上“坠落”而非向下。在[[{{tr|multiplayer}}]]下，这只会影响使用{{tr|item}}的玩家。{{tr|Gravity}}可以通过同一按键再次改回正常，且可以（在{{tr|Gravitation Potion}}的{{tr|buff}}激活或装备{{tr|Gravity Globe}}时）无花费、无冷却地自由切换。

{{tr|reverse gravity}}时值得注意的特性如下：
* 逆转{{tr|gravity}}会重置玩家的[[{{tr|Fall damage}}|下落距离]]。可以在落地前快速地逆转{{tr|gravity}}来避免在长下落中受到{{tr|fall damage}}。
* 在逆转{{tr|gravity}}时碰到{{tr|map}}顶部不会使玩家受到伤害，而是会使得{{tr|gravity}}变回正常。这不会重置[[{{tr|wings}}]]和[[{{tr|Rocket Boots}}]]的飞行计时器。
* [[{{tr|Wisp in a Bottle}}]]，及其控制，在{{tr|reverse gravity}}下会保持颠倒。
* {{tr|reverse gravity}}时可以[[{{tr|Fishing}}]]，{{tr|fishing bobber}}会向上“坠落”到{{tr|water}}中。

在{{tr|reverse gravity}}时也有如下限制：
* [[{{tr|Mounts}}]]无法被召唤。
* [[{{tr|Platforms}}]]不会阻止玩家向上“坠落”，玩家会直接穿过它们。
* [[{{tr|Wings}}]]不能进行滑翔。
* 不能打{{eil|Golf}}；玩家甚至无法进行挥杆。
* {{eil|Kites}}可以使用但仍会保持原有的朝向。
* 不能进入{{eil|Shimmering}}；玩家接触{{tr|Shimmer}}时{{tr|gravity}}会恢复正常。
