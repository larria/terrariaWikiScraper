'''{{tr|Explosive Trap Rod, Explosive Trap Cane, Explosive Trap Staff}}'''是特殊的[[{{tr|Tavernkeep's sentries}}|{{tr|sentry summon weapons}}]]。{{sentry note|name={{tr|Explosive Trap}}|plural=y}}每个{{tr|Explosive Trap sentry}}是一个放置于地面的设施，会在{{tr|enemy}}足够接近时爆炸。它在爆炸后不会消失，而是会保持在召唤位置重复爆炸直至召唤时间耗尽或[[{{tr|Old One's Army}}]]{{tr|event}}结束。在[[{{tr|Tavernkeep}}]]处，{{tr|Explosive Trap Rod}}的售价为 {{iteminfo|Explosive Trap Rod|buy}}，{{tr|Explosive Trap Cane}}的售价为 {{iteminfo|Explosive Trap Cane|buy}}，而{{tr|Explosive Trap Staff}}的售价为 {{iteminfo|Explosive Trap Staff|buy}}。

{{tr|Explosive Trap Rod}}在任何时候都能购买，{{tr|Explosive Trap Cane}}在击败任意[[{{tr|Mechanical Boss}}]] 后可购买，{{tr|Explosive Trap Staff}}在击败[[{{tr|Golem}}]]后可购买。三个不同版本的武器的效果类似，但拥有不同的数据，其中{{tr|Explosive Trap Staff}}是最昂贵的也是最强大的。

直到[[{{tr|Old One's Army}}]][[{{tr|event}}]]成功完成后才能在{{tr|event}}以外使用这些{{tr|weapons}}。在任意{{tr|difficulty}}下成功完成{{tr|event}}后就能允许玩家在{{tr|event}}之外使用{{tr|Rod}}、{{tr|Canes}}、和{{tr|Staves}}。

在{{tr|Old One's Army}}{{tr|event}}持续期间，[[{{tr|Etherian Mana}}]]可被收集，其可用于放置多个{{tr|Explosive Trap}}，{{tr|event}}结束之前会一直存在。{{tr|Etherian Mana}}无法在{{tr|event}}之外使用。

{{tr|Explosive Trap sentries}}可受[[{{tr|Huntress armor}}]]和[[{{tr|Red Riding armor}}]]的{{tr|set bonuses}}增益，这会增加其攻击频率，并使得它们对击中的{{tr|enemies}}造成[[{{tr|Oiled}}]]{{tr|debuff}}，持续 4–8.5 秒。{{source code ref|v=1.4.3.6|method=Terraria.Projectile.StatusNPC}}

== 备注 ==
* 在任何难度中完成[[{{tr|Old One's Army}}]]{{tr|event}}，会为世界以及完成{{tr|event}}的{{tr|character}}设置一个隐藏的属性。任何访问过该世界的{{tr|characters}}（包括新创建的{{tr|characters}}），就可以在任意世界中使用{{tr|Tavernkeep's sentry}}的{{tr|Rods}}、{{tr|Canes}}、和{{tr|Staves}}。完成{{tr|event}}的{{tr|character}}同样也可以在任意世界中使用{{tr|Rods}}、{{tr|Canes}}和{{tr|Staves}}。
* {{tr|Explosive Trap}}在攻击后具有 {{duration|90 ticks}} 的冷却。装备[[{{tr|Huntress armor}}]]后，这一冷却降低至 {{duration|60 ticks}}；装备[[{{tr|Red Riding armor}}]]后，这一冷却降低至 {{duration|30 ticks}}。
* {{tr|sentry}}会探测以其上方第 3 个{{tr|tile}}为中心的 9×9 {{tr|tile}}矩形区域中的{{tr|enemy}}并对其造成伤害。{{source code ref|v=1.4.3.6|method=Terraria.Projectile.AI_138_ExplosiveTrap}}这意味着其范围为向上 7.5 {{tr|tiles}}，向下 1.5 {{tr|tiles}}，向两侧 4.5 {{tr|tiles}}。

== 小贴士 ==
* {{tr|Explosive trap}}会发出微弱的光芒，可用于查看附近周边的洞穴。
* {{tr|Explosive trap}}发射缓慢，射程相对较小。当与[[{{tr|Huntress armor}}]]或[[{{tr|Red Riding armor}}]]结合使用时，它的发射速度会更快，并受益于这些套装的{{tr|summon damage}}提升。在[[{{tr|Old One's Army}}]]{{tr|event}}中，在每个传送门处紧凑放置 3 或 4 个{{tr|Explosive trap}}可以在{{tr|enemies}}离开伤害性区域之前杀死除了[[{{tr|Ogre}}]]以外的所有{{tr|enemies}}。
