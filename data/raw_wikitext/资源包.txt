'''{{tr|Resource Packs}}'''是附加文件，允许玩家自定义游戏的{{tr|sprites}}贴图、[[{{tr|music}}]]、音效和翻译。

== 安装{{tr|Resource Pack}} ==
[[File:Resource Packs menu.png|thumb|right|360px|{{tr|The Resource Packs menu}}。]]
{{tr|Resource Packs}}可以通过将其放置在 <code>ResourcePacks</code> 文件夹中来安装，或通过 Steam 创意工坊下载安装，此时它们会自动安装和更新。

<code>ResourcePacks</code> 文件夹可以通过按游戏内的{{tr|Resource Packs menu}}中的“{{gameText|LegacyInterface.110}}”选项来访问。此文件夹是在启动游戏时生成的。
* 在 Windows Vista 及之后上，是 <code>%USERPROFILE%\Documents\My Games\Terraria\ResourcePacks</code>
** 在 Windows XP 上，是 <code>%USERPROFILE%\My Documents\My Games\Terraria\ResourcePacks</code>
* 在 Mac 上，打开 Finder 并按上方的“Go”。然后按“Go to folder...”。输入 <code>~/Library/Application Support/Terraria/ResourcePacks</code>
* 在 Linux 上，是 <code>$XDG_DATA_HOME/Terraria/ResourcePacks</code>
** 通常是 <code>~/.local/share/Terraria</code>

已安装的{{tr|Resource Packs}}可以在 [[创意工坊中心]] 中启用，或者在主菜单中选择“{{gameText|UI.ResourcePacks}}”选项，在这里可以将资源包在“可用包”和“已启用包”之间移动。

== 创建{{tr|Resource Pack}} ==
=== 内容 ===
{{tr|Resource packs}}必须有 <code>pack.json</code> 文件，否则，游戏无法识别它。它还可以包含一个 <code>icon.png</code> 文件，用来显示在{{tr|Resource Packs}}菜单。有效的 <code>pack.json</code> 文件大概是下面这样：<syntaxhighlight lang="json">
{
	"Name": "包名称",
	"Author": "作者",
	"Description": "这里放置对包的简要描述，可以用 \n 来输入换行",
	"Version": {
		"major": 1,
		"minor": 0
	}
}
</syntaxhighlight>

==== 示例文件夹结构 ====
* <code>包名</code>
** <code>icon.png (可选)</code>
** <code>pack.json</code>
** <code>Content</code>
***<code>Images</code>
**** <code>Acc_Back_1.png</code>
*** <code>Localization</code>
**** <code>en-US_Content.json</code>
*** <code>Music</code>
**** <code>Music_77.ogg</code>
*** <code>Sounds</code>
****<code>Player_Killed.xnb</code>

==== 图像 ====
图像放在资源包的 <code>Content/Images</code> 文件夹下。它们必须使用和默认图像相同的文件名（除了 <code>png</code> 扩展名）和画布尺寸。默认的《{{tr|Terraria}}》图像位于：
* 对于 Windows 的 Steam 版本，是<code>C:\Program Files (x86)\Steam\steamapps\common\Terraria\Content\Images</code>。
* 对于 Linux 的 Steam 版本，是<code>~/.steam/steam/steamapps/common/Terraria/Content/Images</code>
* 对于 macOS 的 Steam 版本，<code>Library/'Application Support'/Steam/steamapps/common/Terraria/Terraria.app/Contents/MacOS/Content/Images</code>.
* 对于 GOG 版本，是<code>C:\GOG Games\Terraria\Content\Images</code>。
** 对 Mac 用户的提示：要打开或查看 Terraria.app 的内容，右键点击并选择“Show Package Contents”

这些默认图像保存为<code>xnb</code> 文件扩展名。可以用 [https://forums.terraria.org/index.php?threads/tconvert-extract-content-files-and-convert-them-back.61706/ TConvert] 来将默认 xnb 图像提取为 png 文件以供查看、编辑、和保存。

==== 音乐 ====
[[音乐]]文件放在资源包的 <code>Content/Music</code> 文件夹下。它们可以是 <code>wav</code>、<code>mp3</code> 或 <code>ogg</code> 格式。

==== 音效 ====
音效文件放在资源包的 <code>Content/Sounds</code> 文件夹下。它们应当是 <code>xnb</code> 格式。

==== 本地化 ====
本地化文件放在资源包的 <code>Content/Localization</code> 文件夹下。它们可以是 <code>csv</code> 或 <code>json</code> 格式。

{{float file box|[[File:DefaultResourcePackIcon.png]]|{{tr|Resource Packs}}的默认图标。}}

* 如果{{tr|Resource Pack}}中没有 <code>icon.png</code>，将改用默认的占位图标。
