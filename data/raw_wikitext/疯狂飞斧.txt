'''{{tr|Possessed Hatchet}}'''是一个[[{{tr|Hardmode}}]]、[[{{tr|Golem}}]]后的[[{{tr|Boomerangs}}|{{tr|boomerang}}]]，有 {{chance|1/7}} 几率掉落自[[{{tr|Golem}}]]。投出后，它会自动导向 50 格<!-- 800 pixels -->以内的[[{{tr|enemies}}]]。{{source code ref|v=1.4.4.9|method=Terraria.Projectile.AI|t=相关部分为 <code>aiStyle == 3</code> 代码段。}}当附近没有{{tr|enemies}}时，它会在飞行 25 格后返回玩家处。和大部分{{tr|boomerangs}}不同，玩家无需等待{{tr|Possessed Hatchets}}返回即可投出更多的{{tr|Possessed Hatchets}}，同时投出的{{tr|Possessed Hatchets}}数量没有限制。{{penetrate|auto=182}}{{source code ref|v=1.4.4.9|method=Terraria.Projectile.SetDefaults}}这和大部分{{tr|boomerangs}}不同，后者在返回时可以击中无限数量的{{tr|enemies}}。

{{modifier|meleealt}}

== 备注 ==
* {{tr|The Possessed Hatchet}}会自动追踪所有[[{{tr|worm enemies}}]]的尾部{{tr|segment}}。
* 和所有{{tr|homing weapons}}一样，在 [[{{tr|PvP}}]] 期间{{tr|the Possessed Hatchet}}不会自动追踪玩家。
* 如果玩家在{{tr|Possessed Hatchet}}被投出时死亡，{{tr|Possessed Hatchet}}会悬浮在死亡地点处，玩家重生后它会返回玩家处。

== 小贴士 ==
* {{tr|The Possessed Hatchet}}对抗[[{{tr|Crawltipede}}]]相当有效，因为它总是会瞄准其尾部。

== 花絮 ==
* 虽然很像{{tr|axe}}，它并不能用来砍[[{{tr|tree}}]]。
