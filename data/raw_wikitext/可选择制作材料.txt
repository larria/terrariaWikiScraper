'''{{tr|alternative crafting ingredient}}'''是可用于替代{{tr|crafting}}[[{{tr|Recipes}}|{{tr|recipe}}]]中另一个{{tr|item}}的[[{{tr|item}}]]。在本 wiki 中，在{{tr|recipe}}章节，替代品会在{{tr|recipe}}中以“''任意&lt;组名&gt;''”的形式或在“''或''”之后列出所有可使用的替代{{tr|item}}。下文列出了所有可能的{{tr|alternative crafting ingredient}}。注意虽然这些物品在很多{{tr|recipes}}中是可替代的，但并非所有{{tr|recipes}}均如此。具体哪些{{tr|recipes}}中是可替代的请参考此物品的页面。

== 编组的{{tr|Items}} ==
有若干个物品组。{{source code ref|v=1.4.3.2|method=Terraria.Recipe.SetupRecipeGroups / Terraria.Recipe.useIronBar / Terraria.Recipe.usePressurePlate / Terraria.Recipe.useSand / Terraria.Recipe.useWood}}可以“混合”来自同一组的物品以节省{{tr|material}}。这意思是，比如，[[{{tr|Chest}}]]可以用 8 个{{tr|Wood}}、1 块{{tr|Iron Bar}}、和 1 块{{tr|Lead Bar}}来制作，尽管玩家并没有足够数量的单独的{{tr|Iron bar}}或{{tr|Lead bar}}。

它们在{{tr|recipes}}的{{tr|ingredients}}表中以“任意&lt;组名&gt;”的形式列出。

== 备注 ==
* 替代{{tr|items}}不适用于名称特定于某种分类词组的[[{{tr|armor}}]]或[[{{tr|weapons}}]]。

== 小贴士 ==
* 当玩家没有大量补给时，在同一个{{tr|item}}上使用两种不同但等价的[[{{tr|bars}}]]有助于节省{{tr|bars}}。

== 花絮 ==
* [[{{tr|Dynasty Wood}}]]是唯一一种不能用于“{{tr|Any Wood}}”{{tr|recipes}}的{{tr|wood}}。
* [[{{tr|Grapes}}]]是唯一不能用于“{{tr|Any Fruit}}”{{tr|recipes}}的{{tr|fruit}}。
* 在游戏代码中还有一个“{{gameText|LegacyMisc.37}}{{gameText|LegacyMisc.51}}”分类，但并未被任何{{tr|recipe}}所使用。它包括四种[[{{tr|Lunar Fragments}}]]。{{source code ref|v=1.4.3.2|method=Terraria.Recipe.useFragment}}
