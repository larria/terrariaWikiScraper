'''{{tr|Goblin Army}}'''（也称为'''{{tr|Goblin Invasion}}'''）是一个[[{{tr|invasion}}]][[{{tr|event}}]]，会使得独特的{{tr|goblin enemies}}代替常规{{tr|enemies}}生成。在满足特定条件后，{{tr|Goblin Army}}可能会在任何[[{{tr|dawn}}]]时随机发生。它也可以用[[{{tr|Goblin Battle Standard}}]]手工召唤。

在{{tr|event}}开始后，{{tr|Goblin Army}}会从{{tr|world}}的随机一侧始发（由下列两个[[{{tr|status messages}}]]之一提示：''“{{gameText|LegacyMisc.1}}”''/''“{{gameText|LegacyMisc.2}}”''）并向[[{{tr|world spawn point}}]]移动。{{tr|status message}}''“{{gameText|LegacyMisc.3}}”''会在{{tr|Goblin Army}}到达了{{tr|world spawn point}}时显示。{{tr|Event enemies}}仅能在{{tr|Goblin Army}}的位置附近或在{{tr|town}} [[{{tr|NPCs}}]] 附近生成。''见[[#{{tr|Notes}}|{{tr|Notes}}]]获取细节。''

{{tr|The Goblin Army}}在击杀足够多来自它的{{tr|enemies}}后即完成。在{{eversions|1.3.0.1}}中，会显示进度条来展示{{tr|event}}进度。

击败{{tr|Goblin Army}}至少一次会满足[[{{tr|Goblin Tinkerer}}]] {{tr|NPC}} 在[[{{tr|cavern}}]]{{tr|layer}}中（作为[[{{tr|Bound Goblin}}]]）生成的条件。

== 召唤 ==
如果{{tr|world}}中至少有一名[[{{tr|player}}]]拥有至少 200 最大[[{{tr|health}}]]，就可以用[[{{tr|Goblin Battle Standard}}]]来手工召唤{{tr|Goblin Army}}。{{source code ref|v=1.4.4.9|method=Terraria.Player.ItemCheck_UseEventItems}}如果没有玩家满足此条件或已经有{{tr|invasion event}}激活，{{tr|the Goblin Battle Standard}}将没有效果且不会被消耗。

如果满足了以下条件，{{tr|Goblin Army}}在{{tr|dawn}}（{{time|4:30 AM}}）也会（以一定几率）随机发生：{{source code ref|v=1.4.4.9|method=Terraria.Main.UpdateTime_StartDay}}
* 没有其他[[{{tr|invasion}}]]{{tr|events}}或[[{{tr|Old One's Army}}]]激活。
* 至少摧毁了一个[[{{tr|Shadow Orb}}]]或[[{{tr|Crimson Heart}}]]。
* {{tr|world}}中至少有一名{{tr|player}}拥有至少 200 最大{{tr|health}}。

{{tr|Goblin Army}}的随机发生有不同的几率：
* 若{{tr|Goblin Army}}尚未被打败，几率为 {{chance|1/3}}。
* 若{{tr|Goblin Army}}已被打败至少一次，在[[{{tr|pre-Hardmode}}]]期间的几率为 {{chance|1/30}}，在[[{{tr|Hardmode}}]]期间的几率为 {{chance|1/60}}。

平均而言，在{{tr|pre-Hardmode}}随机的{{tr|Goblin Invasion}}大约每 12 个现实世界小时会发生一次，在{{tr|Hardmode}}中大约每 24 小时会发生一次。

== 完成 ==
{{tr|The Goblin Army event}}在击杀足够多来自此{{tr|event}}的{{tr|enemies}}后即完成。{{tr|The status message}}''“{{gameText|LegacyMisc.0}}”''会在{{tr|event}}完成时显示。

所需击杀数和{{tr|event}}开始时{{tr|world}}中具有至少 200 {{tr|maximum health}}的玩家数量成正比，可以通过下方公式计算：
: <math>\text{击杀数}=80+40\times\text{玩家数量}</math>

例如：
* 在{{tr|singleplayer}}中，所需击杀数为 <math>80+40\times1=120</math>。
* 在[[{{tr|multiplayer}}]]中有 8 个满足条件的玩家时，所需击杀数为 <math>80+40\times8=400</math>。
* 一次{{tr|invasion}}的最高击杀数为 {{formatnum:{{#expr:80+40*255}}}}（255 个玩家）。

大多数{{tr|enemies}}算作 1 次击杀，但某些{{tr|enemies}}可能算作更多或更少。{{source code ref|v=1.4.4.9|method=Terraria.NPC.checkDead}}详细见下表：{{options/set|item|mode=table}}

== 备注 ==
* {{tr|The Goblin Army}}和其他[[{{tr|invasion}}]]{{tr|events}}一样拥有下列特性：
** 提示{{tr|invasion}}开始的{{tr|status message}}（''“{{gameText|LegacyMisc.1}}”''/''“{{gameText|LegacyMisc.2}}”''）在[[{{tr|world size}}|{{tr|small or medium}}]]{{tr|worlds}}内显示一次，在[[{{tr|world size}}|{{tr|large}}]]{{tr|worlds}}内显示两次。
** [[{{tr|Death}}]]或退出{{tr|world}}不会结束{{tr|invasion}}，也不会影响{{tr|invasion}}的进度。
** 当前{{tr|invasion}}的类型、进度、和所需击杀数会在退出{{tr|world}}时保存。
*** 因此，当{{tr|invasion}}发生时没有阻止它的方法。
** {{tr|Goblin Army}}激活时，{{tr|Town}} [[{{tr|NPCs}}]] 无法{{tr|respawn}}。
** {{tr|invasions}}开始于左侧或右侧[[世界边界]]并以每{{tr|tick}}{{tick}} 1 格的速度移动到{{tr|world}}[[{{tr|spawn point}}]]处（或者说每秒 60 格；{{eicons|1.3.0.1}}此速度受时间速度影响，例如：[[{{tr|Journey Mode}}]]时间滑条、[[{{tr|Enchanted Sundial}}]]、[[入睡]]）。{{source code ref|v=1.4.4.9|method=Terraria.Main.UpdateInvasion}}假设{{tr|world spawn point}}位于{{tr|world}}的水平中心，{{tr|invasions}}在{{tr|small, medium, and large}}[[{{tr|world size}}]]内分别需要 {{#expr:4200/2/60 round 0}}、{{#expr:6400/2/60 round 0}}、和 {{#expr:8400/2/60 round 0}} 秒才能到达{{tr|spawn point}}。
** 玩家必须不低于{{tr|surface}}以下 67.5 格且位于{{tr|invasion}}位置的水平方向（两侧）187.5 格以内才能生成{{tr|invasion enemies}}。此种情况下，来自{{tr|Goblin Army}}的{{tr|enemies}}会完全覆盖当前环境中其他的[[{{tr|NPC spawning}}|{{tr|enemy}}生成]]。{{source code ref|v=1.4.4.9|method=Terraria.NPC.SpawnNPC}}
*** 如果{{tr|invasion}}到达了{{tr|world}}水平中心的 5 格以内的位置，则当玩家位于任何{{tr|town NPC}} 水平方向（两侧）187.5 格以内时也可以通过水平范围检测。{{tr|The town NPC}} 的垂直位置不会被检测。此种情况下，来自{{tr|Goblin Army}}的{{tr|enemies}}只有 {{chance|2/3}} 几率覆盖其他{{tr|enemy}}生成。
** {{tr|An invasion}}不会阻止大多数其他{{tr|events}}发生（如[[{{tr|Blood Moon}}]]、[[{{tr|Solar Eclipse}}]]、或{{tr|night}}的随机 [[{{tr|Boss}}]] 生成）。不过，非 {{tr|boss enemy}}的生成仍然可能被{{tr|invasion enemy}}生成覆盖。
* {{eil|Goblin Warlock}}在[[{{tr|Hardmode}}]]开始时会立即开始生成，即使{{tr|Goblin Army event}}是在[[{{tr|pre-Hardmode}}]]开始的。

== 小贴士 ==
{{strategies|指南:{{tr|Goblin Army}}攻略}}
* 完成此{{tr|event}}至少一次对于推进游戏至关重要，因为它让[[{{tr|Goblin Tinkerer}}]]变为可用，他提供[[{{tr|reforge}}]]功能、[[{{tr|Tinkerer's Workshop}}]]、和[[{{tr|Rocket Boots}}]]。
* [[{{tr|Goblin Peon}}]]会打破[[{{tr|doors}}]]而非打开它们。如果此{{tr|door}}属于包含[[{{tr|Bed}}]] 或 [[{{tr|NPC}}]] 的[[{{tr|House}}]]，这会分别导致{{tr|spawn point}}被重设或 {{tr|NPC}} 搬走。
** {{tr|Goblin Peons}}无法摧毁已经打开的{{tr|doors}}。因此，玩家可能会想要有意地将周围所有{{tr|doors}}打开来防止这邪恶问题。
* 穿透性{{tr|items}}，比如[[{{tr|spears}}]]和[[{{tr|flails}}]]可以同时击中多个{{tr|goblins}}。
* 入侵期间掉落的[[{{tr|Spiky Ball}}]]能有效地击退更多{{tr|goblins}}，因为他们中的大部分都遵循基本的{{tr|Fighter AI}} 并会无脑地穿过它们。
* （和所有{{tr|events}}相同，）[[{{tr|Mediumcore}}]]{{tr|players}}应当小心，因为如果死在了{{tr|spawn point}}附近，那么重新取回{{tr|items}}可能极为艰难甚至几乎不可能，因为这里已经被生成的{{tr|enemies}}占据。在{{tr|Hardmode}}中这可能更糟，因为[[{{tr|Goblin Warlock}}]]可以快速飞行、穿墙射击、并且能够造成很高的{{tr|contact damage}}；这让在重生后收集{{tr|items}}难度极大。

== 另见 ==
* 部分{{tr|goblin}}角色不是{{tr|Goblin Invasion}}的一部分：
** [[{{tr|Goblin Tinkerer}}]]，一个仅在至少打败一次{{tr|Goblin Army}}后出现的{{tr|town NPC}}。
** [[{{tr|Goblin Scout}}]]，一个掉落[[{{tr|Tattered Cloth}}]]且在{{tr|world}}边缘附近生成的{{tr|enemy}}。
