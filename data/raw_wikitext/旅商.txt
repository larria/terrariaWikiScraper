'''{{tr|Traveling Merchant}}'''是一个独特的商贩 [[{{tr|NPC}}]]，他会随机拜访{{tr|player}}的{{tr|town}}并停留一个[[{{tr|Day and night cycle}}|{{tr|day}}]]。他的商品栏是随机的，每次拜访都会不同，出售 4–10 种[[#{{tr|Items sold}}|下表]]列出的不同{{tr|item}}。

想要满足{{tr|Traveling Merchant}}的生成条件，需要已经有至少 2 个其他{{tr|NPC}}已经到达。[[{{tr|Guide}}]]可为其中之一，但不计算[[{{tr|Old Man}}]]和[[{{tr|Skeleton Merchant}}]]。{{source code ref|v=1.3.5.3|method=Terraria.Main.UpdateTime}}他不会在[[{{tr|Solar Eclipse}}]]，[[{{tr|Goblin Army}}]]，[[{{tr|Frost Legion}}]]，[[{{tr|Pirate Invasion}}]]或[[{{tr|Martian Madness}}]]期间生成。{{source code ref|v=1.3.5.3|method=Terraria.WorldGen.SpawnTravelNPC}}他不需要有自己的[[{{tr|house}}]]就能生成，而是生成在其他 {{tr|NPC}}（包括[[{{tr|town pet}}]]）附近，整天围着他们转悠。

一旦满足条件，{{tr|Traveling Merchant}}每{{tr|tick}}{{tick}}将有 1/108000 的几率生成（约合每天22.12%），他将在{{time|4:30 AM}} 和{{time|12:00 PM}} 之间的随机时间点生成。无论{{tr|player}}是否在睡觉，他每{{tr|tick}}生成的几率总是不变。他于当天的{{time|6:00 PM}} 离开，但这只会在他离开屏幕的情况下发生。如果世界中已经没有任何{{tr|player}}（包括死亡状态的），他也将立即消失。

当受到[[{{tr|enemy}}]]的威胁时，{{tr|Traveling Merchant}}将使用[[{{tr|Revolver}}]]自卫。进入[[{{tr|Hardmode}}]]后，他将使用[[{{tr|Pulse Bow}}]]自卫。

== 出售物品 ==
由于使用了复杂的随机化函数且依赖于数个变量，几乎不可能得到{{tr|Traveling Merchant}}出售物品的准确几率。{{source code ref|v=1.4.3.6|method=Terraria.Chest.SetupTravelShop}}，然而，依然可能将{{tr|item}}分组为六个{{tr|rarity}}“{{tr|slot}}”，范围从“常见”到“极其罕见”。请注意，这些{{tr|slot}}与{{tr|Traveling Merchant}}商品栏的{{tr|slot}}并不对应——所有出售的物品可能来自同一{{tr|rarity slot}}。它们只是用来对某一{{tr|item}}的{{tr|rarity}}做一个粗略的、比较性的估计。每个商品栏{{tr|slot}}都有 1/n 的几率出现每个可能会出售的{{tr|item}}，其中 n 是 100 × {{tr|rarity slot}}。该几率受到[[{{tr|luck}}]]影响，更确切地说，受到{{tr|world}}中{{tr|luck}}最高的{{tr|player}}的影响。当多个{{tr|item}}都通过了几率判定，则会选择最后一个通过判定的{{tr|item}}。

在同一{{tr|rarity slot}}中，在下表中排列更高的{{tr|item}}有更高的几率出售，例如，[[{{tr|Companion Cube}}]]比[[{{tr|Water Gun}}]]更有可能出售（''平均而言''），因为它在列表中排列更高。

{{tr|Traveling Merchant}}总会出售一幅{{tr|painting}}，且总是在他商品栏的最后一个{{tr|slot}}里。在[[{{tr|Hardmode}}]]，至少会有一个出售的{{tr|item}}对应{{tr|rarity slot 2}}（不常见）或更稀有。

{{tr|Traveling Merchant}}商品栏中的出售{{tr|item}}数量在每次到达时随机变化。平均出售{{tr|item}}数量为 {{formatnum:{{#var:avg_classic}}}} / {{expert|{{formatnum:{{#var:avg_expert}}}}}}。出售{{tr|item}}数量及其对应的具体概率如下表所示。{{source code ref|v=1.4.4.9|method=Terraria.Chest.SetupTravelShop}}

此外，[[{{tr|painting}}]]也不计入这个数目，因为{{tr|Traveling Merchant}}在所有非{{tr|painting}}外总是额外出售一幅{{tr|painting}}。

“基础”{{tr|item}}数量受{{tr|player}}的[[{{tr|luck}}]]影响。较高的{{tr|luck}}会增加{{tr|Traveling Merchant}}出售超过 6 个“基础”{{tr|item}}的几率，较低的{{tr|luck}}会增加他只出售 4-6 个“基础”{{tr|item}}的几率。但是，{{tr|Luck}}只是在现有的范围内调整几率，因此，{{tr|Traveling Merchant}}总是会出售至少 4 个且至多 10 / {{expert|11}} “基础”{{tr|item}}（在下述条件下可更多），即使{{tr|luck}}特别高或特别低。在[[{{tr|multiplayer}}]]中，{{tr|world}}当前{{tr|luck}}最高的{{tr|player}}的{{tr|luck}}会有影响。

在某些情况下，出售“基础”{{tr|item}}数量会增加：
* 如果当前{{tr|world}}处于[[{{tr|Hardmode}}]]，数目增加 1。
* 如果当前{{tr|world}}已经使用过[[{{tr|Peddler's Satchel}}]]，数目增加 1。
* 在 [[{{tr|Get fixed boi}}]] {{tr|world}}，数目增加 1。
* 在 [[{{tr|Celebrationmk10}}]] {{tr|world}}，数目增加 2。

== 备注 ==
* 生成{{tr|Traveling Merchant}}的判定并不是如[[{{tr|Goblin Army}}]]或其他大部分[[{{tr|event}}]]一样一天执行一次。而是每{{tr|tick}}{{tick}}执行一次。
** 对于每天前半部分的每个{{tr|tick}}（{{time|4:30 AM}} – {{time|12:00 PM}} = 7.5 {{tr|hour}} = 27000 {{tr|tick}}），有 1/108000 的几率生成{{tr|Traveling Merchant}}。因此，{{tr|Traveling Merchant}}每天有：<code>1-(107999/108000)^27000 = 0.2212001339 ≈ 22.12%</code>的几率到达。
** 这意味着如果想要{{tr|Traveling Merchant}}出售的某个特定{{tr|item}}，但他到达后并没有出售该{{tr|item}}，且此时在游戏中为{{time|12:00 PM}} 之前，可以击杀{{tr|Traveling Merchant}}，以等待潜在的二次生成。
* {{tr|Traveling Merchant}}在[[{{tr|event}}]]期间不会离去。
* {{tr|Traveling Merchant}}可以举办[[{{tr|Party}}]]。

== 小贴士 ==
* {{tr|Traveling Merchant}}只有在离开屏幕后才会消失。如果{{tr|player}}在他通常离开的时间点让他停留在屏幕内，他将一直停留，直到他离开屏幕。如果他一直停留在屏幕内，直到第二天黎明，他将在第二天晚上{{time|6:00 PM}}之后离开。然而，这并不会改变他的商品栏。

== 花絮 ==
* 他出售的许多商品都来自于亚洲，通常来自于{{wikipedia|Qing dynasty|清朝}}。
** 他的很多商品可能参考了{{wikipedia|East India Company|东印度公司}}，也可能是参考了在大发现时期中被带到欧洲的商品。
** 他的[[#{{tr|Shimmered form}}|{{tr|Shimmered form}}]]穿着亚洲风格的服饰。
* 他的名字 Willy 很有可能参考了美国剧作家{{wikipedia|Arthur Miller|亚瑟·米勒（Arthur Miller）}} 1949 年写就的剧作{{wikipedia|Death of a Salesman|推销员之死}}，剧中的主角 {{wikipedia|Willy Loman}} 是一个旅行推销员。
* 他的话语“{{gameText|LegacyDialog.328}}”参考了巴基斯坦商人和广播艺术家 Muhammad Shahid Nazir，他因其新奇歌曲{{wikipedia|One Pound Fish|《一磅鱼（One Pound Fish）》}}走红，被称为{{wikipedia|One Pound Fish Man|一磅鱼男（One Pound Fish Man）}}。
* 他的话语“{{gameText|LegacyDialog.327}}”参考了 1992 年的美国动画音乐奇幻电影{{wikipedia|Aladdin (1992 Disney film)|《阿拉丁（Aladdin）》}}。
* 他的话语“{{gameText|LegacyDialog.322}}”参考了一个有关[[{{tr|Angel Statue}}]]是游戏中最没用的{{tr|statue}}的没品笑话；[[{{tr|Merchant}}]]的一句话语也参考了这个。
* 和[[{{tr|Cyborg}}]]、[[{{tr|Zoologist}}]]以及[[{{tr|Princess}}]]一样，{{tr|Traveling Merchant}}是仅有的几个在击败[[{{tr|Moon Lord}}]]后有新{{tr|item}}出售的 [[{{tr|NPC}}]] 之一，他出售的是[[{{tr|Not a Kid, nor a Squid}}]]和[[{{tr|Moonman & Company}}]]。
