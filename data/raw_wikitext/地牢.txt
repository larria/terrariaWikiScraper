'''{{tr|Dungeon}}'''是个巨大的地下[[{{tr|brick}}]]制迷宫，在{{tr|world creation}}之时生成，位于地图的最左侧或最右侧，并且包含独特的[[{{tr|enemies}}]]（大部分为{{tr|skeleton}}主题），其{{tr|spawn rates}}随着玩家深入而提升。它从一个类似城堡的地上入口建筑开始，向下延伸到接近[[{{tr|Underworld}}]]的深度。每个世界会生成一个{{tr|Dungeon}}。要进入{{tr|the Dungeon}}需要先打败[[{{tr|Skeletron}}]]''（见下文的[[#进入|进入]]）''。{{tr|The Dungeon}}可以以三种[[{{tr|Dungeon Brick}}]]颜色之一出现，并有一种或多种[[{{tr|Dungeon Brick Wall}}]]图案，均在{{tr|world creation}}之时随机生成。

在{{eversions|1.4.0.1}}中，有一些道路会被[[{{tr|Cracked Dungeon Bricks}}]]阻挡。这些{{tr|blocks}}看起来和{{tr|Dungeon Brick}}几乎一样，但是有轻微的碎裂，并且在被摧毁时不会掉落{{tr|blocks}}。挖掘一个{{tr|block}}可能会使相邻的{{tr|Cracked Dungeon Bricks}}一起连续破碎，这和[[{{tr|Desert Fossil}}]]的特性相似。当玩家在{{tr|Cracked Dungeon Bricks}}上行走、在其上使用[[{{tr|hooks}}]]、或用[[{{tr|projectile}}]]击中它们时，它们也可能会自发碎裂。使用[[{{tr|Dangersense Potion}}]]可以将这些{{tr|blocks}}突出显示。{{tr|Cracked Dungeon Bricks}}也会覆盖住大型的环绕着{{tr|Spike}}的长方形房间，有时这些房间还会充满{{tr|water}}，因此建议玩家小心探索。

{{tr|The Dungeon}}包含密集且多样的独特[[{{tr|Dungeon Furniture}}|{{tr|furniture}}]]和[[{{tr|Mechanism}}]]，玩家可以将其回收以备日后使用。一些[[{{tr|Dart Trap}}]]和[[{{tr|Pressure Plate}}]][[{{tr|Wire}}|连接在一起]]，给玩家带来了额外的危险。像这样的{{tr|Traps}}有时会出现在地面之上的{{tr|the Dungeon}}入口处。还会出现和[[{{tr|Switch}}]]连接在一起的[[{{tr|Chain Lantern}}]]，玩家可以将其打开和关闭。受缚[[{{tr|Mechanic}}]]也出现于此，与其说话可将其释放。

这里有一些[[{{tr|Locked Gold Chest}}]]，里面既有{{tr|Dungeon}}特有的{{tr|loot}}也有标准的[[{{tr|underground}}]]{{tr|loot}}。它们每个都需要一把[[{{tr|Golden Key}}]]才能打开，钥匙在开{{tr|chest}}的锁时会被消耗掉。还会存在加锁的[[{{tr|Biome Chest}}]]，但它们只在[[{{tr|Plantera}}]]已被打败且玩家已经得到相应的[[{{tr|Biome Key}}]]后才能打开。

在[[{{tr|Plantera}}]]被打败后，{{tr|the Dungeon}}会生成新的更强大的{{tr|enemies}}，它们提供了新的物品。并非所有{{tr|enemies}}都会在所有{{tr|Dungeons}}内生成：对于特定{{tr|Dungeon}}而言，其中生成的特定{{tr|background wall}}类型决定了这里会生成哪些{{tr|Plantera}}后的{{tr|enemies}}。（''参见[[#{{tr|Hardmode}}{{tr|Plantera}}之后的{{tr|Dungeon}}|下文]]''）
[[File:WaterBolt.png|thumb|放置的[[{{tr|Water Bolt}}]]。]]
{{float file box|[[File:Lesser Healing Potion (placed).png]]&#32;[[File:Lesser Mana Potion (placed).png]]|放置的[[{{tr|Lesser Healing Potion}}]]和[[{{tr|Lesser Mana Potion}}]]}}

{{tr|the Dungeon}}中有许多{{tr|shelves}}，上面有[[{{tr|Books}}]]、[[{{tr|Lesser Healing Potion}}]]、和[[{{tr|Lesser Mana Potion}}]]，偶尔还会有[[{{tr|Water Bolt}}]]，它呈现为一本放置状态的黄条纹蓝皮书。[[{{tr|Life Crystal}}]]无法在{{tr|Dungeon}}中生成。

在{{eversions|Lunatic Cultist|short=y}}上，玩家在世界中打败[[{{tr|Golem}}]]和[[{{tr|Skeletron}}]]后，在{{tr|Dungeon}}入口处会生成一组 4 名[[{{tr|Cultist}}]]，击杀他们会召唤[[{{tr|Lunatic Cultist}}]]，击杀{{tr|the Lunatic Cultist}}将发动[[{{tr|Lunar Events}}]]，这是游戏的终极[[{{tr|event}}]]。

== 进入 ==
要想畅通无阻地进入{{tr|the Dungeon}}，必须先打败[[{{tr|Skeletron}}]]。与守在{{tr|Dungeon}}入口处的[[{{tr|Old Man}}]]交谈可以召唤出{{tr|Skeletron}}。在[[{{tr|night}}]]，交谈时会显示“{{gameText|LegacyInterface.50}}”选项，选择它会让{{tr|Old Man}}变成{{tr|Skeletron}}。在[[{{tr|day}}]]和{{tr|the Old Man}}交谈不会允许玩家和{{tr|Skeletron}}战斗。打败{{tr|Skeletron}}后，[[{{tr|Clothier}}]] [[{{tr|NPC}}]] 变得可用，当有合适的空[[{{tr|house}}]]可用时他会生成。

在打败{{tr|Skeletron}}之前，{{tr|The Dungeon}}可以进入但不能完全探索。在打败它之前，在{{tr|Dungeon}}中前进到 -3 [[{{tr|depth}}]]时会召唤一个或多个[[{{tr|Dungeon Guardian}}]]，它们一击就能杀掉大部分玩家。此{{tr|Dungeon Guardians}}几乎是无敌的，并且击杀它们也''不会''得到进入{{tr|Dungeon}}的权限，打败{{tr|Dungeon Guardian}}唯一能得到的物品是[[{{tr|Bone Key}}]]。

要判断是否已满足生成{{tr|Dungeon Guardians}}的要求，玩家可以听{{tr|music}}是否变为独特的{{tr|Dungeon}}配乐，或者如果装备了[[{{tr|Depth Meter}}]]，查看其读数是否是“{{gameText|GameUI.DepthLevel}}”或“{{gameText|GameUI.Depth|x_0=X}} {{gameText|GameUI.LayerUnderground}}”。{{tr|Dungeon}}入口的位置高度差异很大，但总是在零深度以上，这使得其入口建筑物和其下 1–2 个屏幕深度通常是安全不受{{tr|Dungeon Guardians}}影响的。

== {{tr|Hardmode}}{{tr|Plantera}}后的{{tr|Dungeon}} ==
当世界中[[{{tr|Plantera}}]]首次被打败时，会出现[[{{tr|status message}}]]''“{{gameText|LegacyMisc.33}}”''，意味着现在在{{tr|Dungeon}}中将会出现新的[[{{tr|enemy}}]]。它们通常比普通的{{tr|Dungeon}}{{tr|enemies}}困难得多，并会掉落新的强力物品。另外，也将可以打开存在于{{tr|Dungeon}}中的[[{{tr|Biome Chest}}]]。

在一个{{tr|Dungeon}}区域中所[[{{tr|NPC spawning}}|{{tr|spawn}}]]的特定{{tr|Plantera}}后的{{tr|enemies}}取决于此处的[[{{tr|background walls}}]]类型，{{tr|enemies}}类型也与出现在此区域的自然放置的[[{{tr|Faction Flag}}]]相对应。单个{{tr|Dungeon}}可以在不同部分包含多种{{tr|background wall}}类型，也可以只有一种。{{tr|a Dungeon}}是否包含不同类型的{{tr|wall}}只能通过探索{{tr|the Dungeon}}来确定。不同类型的{{tr|wall}}有相应的“常见”{{tr|enemies}}，但其他类型的{{tr|Dungeon enemies}}总有较小的几率生成，该几率会因为附近[[{{tr|luck}}]]较高的玩家的存在而增加。例如，{{tr|Skeleton Snipers}}仍然可以罕见地生成在{{tr|Dungeon Brick Walls}}的区域中，即使附近没有{{tr|Slab Walls}}也是如此。

== 备注 ==
[[File:Surface Dungeon Chest.png|thumb|一个所在位置足够高从而能在打败{{tr|Skeletron}}之前拾取的{{tr|Chest}}。]]

* {{anchor|Dungeon point}}{{anchor|{{tr|Dungeon}}点}}在内部，会选择{{tr|Dungeon}}入口处的一个{{tr|tile}}（“{{tr|Dungeon}}点”）作为[[{{tr|Old Man}}]]和{{eil|Mysterious Tablet}}（以及[[{{tr|Cultists}}]]）的{{tr|spawning tile}}。这个{{tr|tile}}是{{tr|Old Man}}在{{tr|night}}期间所站位置下方两个{{tr|blocks}}中左边的那个。
* 相对于世界的水平中心点，{{tr|The Dungeon}}总是位于[[{{tr|Jungle}}]]的对面，和[[{{tr|Snow}}]]{{tr|biome}}的同一侧。{{source code ref|v=1.4.2.3|method=Terraria.WorldGen.GenerateWorld|t=相关的 world generation pass 被称为“Reset”。}}
* 玩家视为在{{tr|the Dungeon}}内的条件是深度为 -3 英尺（{{tr|surface}}下方 1.5 {{tr|tile}}）或以下、站在[[{{tr|Unsafe wall}}|不安全]]的[[{{tr|Dungeon Brick Walls}}]]前面，且[[{{tr|生物群系的存在要求}}]]区域内至少有 250 个[[{{tr|Dungeon Brick}}]]。{{source code ref|v=1.4.4.9|method=Terraria.Player.UpdateBiomes}}如果想要引出{{tr|Dungeon Guardian}}或避免与其遭遇，这个知识点会很有用。
* 对于大多数{{tr|Dungeons}}，除了其可见的{{tr|background walls}}之外，其前景{{tr|blocks}}背后也有{{tr|brick wall}}。如果这些{{tr|blocks}}被挖掉且{{tr|Plantera}}已被打败，与{{tr|brick walls}}关联的{{tr|enemies}}同样可能会生成。
* {{tr|Dungeon}}中最初几个{{tr|chests}}（由{{tr|Dungeon}}入口开始）中的独特{{tr|loot}}不通过解锁也能够容易地猜出来。如果第一个{{tr|chest}}在 -100 英尺深度以上，它将会是一个{{tr|wooden chest}}并且含有一把{{tr|golden key}}，否则它将含有{{tr|muramasa}}{{eicons|1.4.0.1}}/{{tr|Shadow Key}}{{eicons|1.4.0.1|invert=y}}。下一个{{tr|chest}}中会包含{{tr|Cobalt Shield}}{{eicons|1.4.0.1}}/{{tr|Muramasa}}{{eicons|1.4.0.1|invert=y}}，等等。
** 这源于{{tr|Dungeon chest loot}}生成的方式；每个{{tr|chest}}中的独特{{tr|item}}是按既定顺序排列的。实际情况下，由于{{tr|Dungeon}}会分为多个分支路线，很难跟从这一顺序（来寻找特定{{tr|item}}）。然而，由于{{tr|Dungeon}}入口是单一的一条路径，这里的宝藏是可预测的。宝藏的顺序为：{{tr|Shadow Key}}{{eicons|1.4.0.1|invert=y}}、{{tr|Muramasa, Cobalt Shield, Aqua Scepter, Blue Moon, Magic Missile, Valor, Golden Key, Handgun}}。
** 注意，在{{eversions|1.4.0.1}}中，{{tr|Shadow Key}}会与其他的独特{{tr|Dungeon treasure}}''一起''生成，而非单独生成，因此在这些版本中需要被排除出{{tr|loot}}列表。
** 不过，这并不意味着{{tr|Dungeon}}中遇到的第一个{{tr|Gold Chest}}总会包含{{tr|Muramasa}}，这取决于{{tr|world}}是怎样生成的。
* {{tr|Dungeons}}在{{eversions|1.3.0.1|invert=y|short=y}}版的[[《{{tr|Terraria}}》]]上要小很多。
* 在[[{{tr|Skeletron}}]]之前，[[{{tr|Dungeon Guardian}}]]有可能在玩家处于 -3 深度以下且在{{tr|Dungeon}} ''之外''时生成，虽然可能性非常小。这只会在玩家极度接近{{tr|Dungeon}}，接触或者紧挨着外墙时才会发生。
* 有可能会在{{tr|surface}}以上生成一个或多个[[{{tr|Chest}}]]，可以拿走而不会触发{{tr|Dungeon Guardians}}生成。但是，其中不会包含独特而宝贵的{{tr|Dungeon}}特有物品。
* {{eicons|1.4.0.1|invert=y}}在{{tr|surface}}以上有几率生成{{tr|water bolt}}，使得玩家在游戏早期能够获得一件强力{{tr|weapon}}。
