'''{{tr|Luminite Bullet}}'''是一种[[{{tr|Hardmode}}]]、[[{{tr|Moon Lord}}]]后的[[{{tr|bullets}}|{{tr|bullet}}]]，由[[{{tr|Luminite Bar}}]]制作。{{penetrate|auto=638|penalty=4%}}这可以将其伤害降低至 0，使其无法造成{{tr|damage}}但仍然继续行进。它会在 {{duration|100ticks}}后{{source code ref|v=1.4.4.9|method=Terraria.Projectile.SetDefaults}}或击中{{tr|solid block}}后消失。

{{tr|Luminite Bullets}}在击中目标时会产生亮蓝色的粒子。

== 备注 ==
* {{tr|The Luminite Bullet}}具有独特的穿透机制。它对目标造成 -1 {{tr|ticks}}的[[{{tr|local invincibility frame}}]]{{source code ref|v=1.4.4.9|method=Terraria.Projectile.Damage}}，这意味着：
** 单个{{tr|Luminite Bullet}}只能对每个目标造成一次伤害。
** {{tr|Luminite Bullets}}的{{tr|Damage}}不会阻止任何其他{{tr|piercing projectiles}}伤害目标。
** {{tr|Luminite Bullets}}依然无法击中由于其他事情（如造成[[{{tr|global invincibility frame}}]]的{{tr|item}}）而''已经无敌''的目标。这使得{{tr|Luminite Bullets}}能和其他{{tr|Luminite Bullet}}使用者协作，但和其他穿透性武器一起使用效果较差。
* {{tr|Luminite Bullets}}比[[{{tr|High Velocity Bullet}}]]略慢。这在非常高的屏幕分辨率下最为明显。

== 小贴士 ==
* {{tr|Luminite Bullets}}由于其穿透效果、高{{tr|base damage}}、高{{tr|velocity}}而对[[{{tr|Sniper Rifle}}]]起到了很好的补充作用，进一步提高了{{tr|the Sniper Rifle}}的伤害。
* 尽管有所有{{tr|bullets}}中最高的{{tr|base damage}}，但其穿透能力让{{tr|Luminite Bullets}}更适合群体控制而非单体目标。像[[{{tr|S.D.M.G}}]]这样的快速开火的{{tr|gun}}用在这非常好。

== 花絮 ==
* 假设{{tr|weapon}}不造成[[{{tr|critical strike}}]]，一发{{tr|Luminite Bullet}}搭配没有其他{{tr|damage}}加成的[[{{tr|Annoying}}]][[{{tr|Minishark}}]]时最多能够伤害 26 个{{tr|enemies}}，或者搭配具有所有可能{{tr|damage}}加成的[[{{tr|Ruthless}}]][[{{tr|Sniper Rifle}}]]时最多能够伤害 101 个{{tr|enemies}}。
* {{tr|Luminite Bullet}}具有所有[[{{tr|bullets}}]]中最高的{{tr|base damage}}。
* {{tr|Luminite Bullets}}是唯一不需要用[[{{tr|Musket Ball}}]]或[[{{tr|Empty Bullet}}]]制作的{{tr|bullets}}。
