[[File:BiomeBannerForest.png|center]]

'''{{tr|Forest}}'''是一种[[{{tr|Pure biomes}}|纯净]]的[[{{tr|biome}}]]，构成了[[{{tr|surface}}]]{{tr|world}}的绝大部分，并且通常情况下玩家会初始[[{{tr|spawn}}]]在这里。{{tr|forest}}的大体景观包括大量的[[{{tr|grass}}]]、许多[[{{tr|tree}}]]、大小不同的若干含[[{{tr|water}}]]湖泊和池塘，且偶尔会有具有洞穴入口的小山。{{tr|Forest}}经常会包含一棵或更多的[[{{tr|Living Wood}}|{{tr|Living Tree}}]]，后者有时会带有{{tr|underground treasure room}}。

在世界两侧各六分之一的范围内，{{tr|The Forest}}发生了显著的变化。会开始生成[[{{tr|Goblin Scout}}]]，它相当少见，并且会掉落[[{{tr|Tattered Cloth}}]]。也有相当低的几率生成[[{{tr|King Slime}}]] [[{{tr|Bosses}}|{{tr|boss}}]]，很容易就让毫无戒心的玩家目瞪口呆且不知所措。{{eil|Enchanted Sword Shrine|s}}只能在世界两侧各三分之一的范围内找到。

== 山洞 ==
[[File:Mountain Cave map view.png|thumb|山洞在地图上的样子。]]

'''山洞'''是通常可在{{tr|Forest biome}}中找到的结构。它是一座高高的[[{{tr|Dirt}}]]山，其中具有通向[[{{tr|underground}}]]的弯曲洞穴，可以作为游戏早期的玩家轻松前往{{tr|underground}}的选择。[[{{tr|Pots}}]]和木制的[[{{tr|Chest}}]]经常可在洞穴中发现。

== {{tr|NPC happiness}} {{eicons|Happiness}} ==
{{living preferences}}

== 备注 ==
* 一般而言，{{tr|the Forest}}是游戏中最容易的{{tr|biome}}，因为在这里所碰到的{{tr|enemies}}只要拿到了一些基本的装备就能很轻易打败。玩家的第一个基地通常会建在这里，因为很快就能找到诸如[[{{tr|Clay}}]]、[[{{tr|Stone}}]]、和[[{{tr|Wood}}]]等建筑材料，而{{tr|enemy}}{{tr|the spawn rate}}通常很低，且最初的生成点一般都会在{{tr|the Forest}}中。
* 低级别的[[{{tr|ore}}]]有小几率会在{{tr|forest}}的{{tr|surface}}生成。可能出现的各种{{tr|ores}}包括[[{{tr|Copper Ore}}|{{tr|Copper}}]]或[[{{tr|Tin Ore}}|{{tr|Tin}}]]、[[{{tr|Lead Ore}}|{{tr|Lead}}]]或[[{{tr|Iron Ore}}|{{tr|Iron}}]]、有时还有[[{{tr|Silver Ore}}|{{tr|Silver}}]]或[[{{tr|Tungsten Ore}}|{{tr|Tungsten}}]]。{{tr|surface}}上有时还会有小片{{tr|Clay Block}}和{{tr|Stone Block}}。
* [[{{tr|Chests|link=y}}#{{tr|Surface loot}}|{{tr|Surface chests}}]]散落在{{tr|Forest}}各处，内含许多特殊的开局物品。
* 在[[{{tr|Hardmode}}]]中，此{{tr|biome}}会略有强化，在{{tr|night}}会有新{{tr|enemies}}生成。
* {{eicons|1.4.0.1}}在世界生成过程中，每棵{{tr|regular Forest tree}}都有 {{chance|1/20}} 几率被替换为[[{{tr|Sakura tree}}]]和[[{{tr|Yellow Willow tree}}]]。
* {{eicons|1.4.0.1}}{{tr|Forest}}中会生成[[{{tr|Fallen Logs}}]]，{{tr|night}}时会有[[{{tr|Fairies}}]]在此处生成。当用[[{{tr|Axes}}|{{tr|Axe}}]]破坏它们时会掉落一些{{tr|Wood}}，而将其全数破坏能够永久性地阻止{{tr|surface Fairies}}生成（但依然可以通过[[{{tr|Trees|link=y}}#{{tr|Shaking}}|{{tr|shaking trees}}]]来生成它们）。

== 小贴士 ==
* 常规的{{tr|Forests}}会被[[{{tr|The Corruption}}]]、[[{{tr|The Crimson}}]]、和[[{{tr|The Hallow}}]]的[[{{tr|spread}}]]消除。被转化的{{tr|forests}}可以用购买自[[{{tr|Dryad}}]]的[[{{tr|Thrown Powder}}|{{tr|Purification Powder}}]]或购买自[[{{tr|Steampunker}}]]的更强的{{tr|Hardmode}}对应物[[{{tr|Clentaminator}}]]和{{tr|Green Solution}}来恢复到纯净形态。

== 花絮 ==
* 截至 {{eil|1.4.0.5}}，{{tr|the forest}}拥有最多音乐曲目，算上来自[[《{{tr|Terraria: Otherworld}}》]]的那些共有五首。
* {{eicons|Happiness}} 根据 [[{{tr|NPC}}]] [[{{tr|happiness}}]]来说，{{tr|the Forest}}是最受欢迎的{{tr|biome}}：只有两个 {{tr|NPCs}} 不喜欢它，而有 4 个 {{tr|NPCs}} 喜欢它。单就数字来说，这要好于所有其他{{tr|biome}}。{{tr|The Hallow}}是唯一能接近这个数字的{{tr|biome}}。
