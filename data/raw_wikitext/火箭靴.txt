'''{{tr|Rocket Boots}}'''是一件[[{{tr|accessory}}]]，能让玩家总共飞行 {{duration|70ticks}}。{{source code ref|name=update|v=1.4.4.9|method=Terraria.Player.Update}}需要在半空中按住 {{key|jump}} 以激活它。飞行时，它会发出喷气动画，会产生较亮的光。{{tr|Rocket Boots}}以 {{iteminfo|Rocket Boots|buy}} 购买自 [[{{tr|Goblin Tinkerer}}]]。

飞行时间会在玩家落在任何实心物体，或用[[{{tr|hooks}}]]抓住表面时立即重置。

如果装备了任何[[{{tr|extra jump}}]]{{tr|accessories}}，{{tr|extra jump}}效果会率先生效；此后在半空中按住跳跃按键会激活{{tr|Rocket Boots}}。

* 飞行时间是累计的，这意味着可以把飞行时间拆开来一段一段使用。
* 飞行时间无法与{{tr|its upgrades}}叠加（[[{{tr|Spectre Boots}}]]、[[{{tr|Lightning Boots}}]]等等）。
* {{tr|Rocket Boots}}的最大上升速度为 38 {{mph}}。
* 当和[[{{tr|wings}}]]搭配时，{{tr|Rocket Boots}}的飞行会被禁用；它会改为直接使{{tr|wings}}的飞行时间增加 {{duration|42ticks}}。<ref name="update"/>
* 如果装备了全套的[[{{tr|Shadow armor}}]]、[[{{tr|Ancient Shadow armor}}]]、或[[{{tr|Ancient Cobalt armor}}]]，喷气效果会改为与之搭配的紫色。{{source code ref|v=1.4.4.9|method=Terraria.Player.PlayerFrame}}

== 小贴士 ==
* 玩家应当注意使用{{tr|Rocket Boot}}后会掉落的高度。（使用{{tr|Rocket Boot}}时）很容易就能达到足以死于[[{{tr|fall damage}}]]的高度，尤其是搭配了其他强化跳跃的{{tr|accessories}}时。
** 可以留下部分飞行时间来避免{{tr|fall damage}}。
