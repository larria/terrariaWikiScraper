'''{{tr|Items}}'''是一些实体，是《[[{{tr|Terraria}}]]》的基本组件之一。玩家可以通过使用、消耗、或放置来与大多数{{tr|item}}互动。{{tr|Items}}总是处于两种形态之一：“掉落”，即，是[[{{tr|world}}]]世界的一部分；或在与玩家相关的、[[{{tr|NPCs}}|{{tr|NPC vendor}}]]的、或[[{{tr|storage item}}]]的[[{{tr|inventory}}]]中。掉落的{{tr|items}}通常可以被玩家拾取进入其{{tr|inventory}}中，并且通常可以再次从{{tr|inventory}}中丢弃出去。

大多数{{tr|items}}可以根据某些共通的特征进行分组，例如[[{{tr|armor}}]]{{tr|items}}、[[{{tr|blocks}}]]、或[[{{tr|weapons}}]]。关于完整的分类，参见[[:分类:物品]]。

== 获取 ==
在《{{tr|Terraria}}》中，可以通过多种方式获得{{tr|items}}。包括
* 在实体掉落它们（例如，打败[[{{tr|enemies}}]]、挖掉[[{{tr|blocks}}]]或[[{{tr|furniture}}]]）后拾取之；
* 从另一个{{tr|inventory}}（例如，{{tr|NPC}} 商贩或[[{{tr|Chest}}]]）转移过来；
* 作为其他流程的一部分直接放置在玩家的{{tr|inventory}}中（例如[[{{tr|crafting}}]]或被[[{{tr|Angler}}]]等 {{tr|NPCs}} 奖励）。

== 拾取 ==
当{{tr|item}}处于“掉落”状态时，任何接触到此{{tr|item}}的玩家都会拾取之，并自动放入其{{tr|inventory}}中（如果有足够空间，且{{tr|inventory}}中没有{{eil|Encumbering Stone}}）。另外，所有玩家都会将一定范围内的所有物品牵引（“拉”）向自身以制造接触。此范围称为''拾取范围''。默认情况下，它是一个以玩家为中心的 5.25×5.25 格的矩形区域（即向四方向都延伸 2.625 格）。某些效果会提升某些{{tr|item}}的拾取范围：{{source code ref|v=1.4.1.2|method=Terraria.Player.GetItemGrabRange}}

== 堆叠 ==
在[[{{tr|inventory}}]]中，{{tr|items}}可以在单个栏位中堆叠至特定上限数量。

在{{eversions|1.4.4|short=y}}中，大多数物品都可以堆叠至 9999 个，但有少数例外（如{{tr|equipment}}和[[{{tr|Quest fish}}]]）。

在{{eversions|1.4.4|invert=y|short=y}}中，不同类型的物品有不同的上限，如下表所示。这只是一个整体概述，有例外情况。各个物品的最大堆叠大小请参阅物品的单独页面。

== 消失 ==
处于“掉落”状态的物品会在退出世界时消失。从那个时间点起就不可能再重新找回它们。如果世界中“掉落”状态的物品超过 400 个，则也会开始有物品消失。在这种情况下，新物品加到屏幕上时，掉落时间最久的物品会消失。
