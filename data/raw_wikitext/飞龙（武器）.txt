'''{{tr|Flying Dragon}}'''是一把[[{{tr|Hardmode}}]]，[[{{tr|Golem}}]]后的[[{{tr|sword}}]]。它每 20 {{tr|ticks}}{{tick}}{{source code ref|v=1.4.4.9|method=Terraria.Item.SetDefaults4 / Terraria.Player.ItemCheck_OwnerOnlyCode / Terraria.Player.ApplyItemTime}}会发射一个红色新月形的{{tr|Hearty Slash}}{{tr|projectiles}}，{{tr|projectile}}会穿过实体[[{{tr|blocks}}]]并发光。{{tr|Hearty Slash}}{{tr|projectiles}}可以存在一秒且能穿透四个[[{{tr|enemy}}]]，在击中第五个{{tr|enemy}}或移动了 60 {{tr|tile}}时消失。

{{modifier|melee}}

== 备注 ==
* 刀刃的{{tr|hitbox}}远远超过了刀刃{{tr|sprite}}本身，在水平{{tr|range}}只比[[{{tr|Breaker Blade}}]]少约一个{{tr|pixel}}，但在垂直{{tr|range}}内损失相当大。
* {{tr|Hearty Slash}}的{{tr|projectiles}}{{tr|speed}}受{{tr|melee speed}}{{tr|bonuses}}影响极小，可几乎忽略。
* 它的{{tr|projectile}}{{tr|speed}}为{{mph|16 pixels}}-{{mph|22.6267 pixels}}，取决于其发射角度。

== 小贴士 ==
* 由于其可以打出发光且可穿墙的光波，因此此{{tr|sword}}适用于探索和寻找洞穴，就像[[{{tr|Death Sickle}}]]一样。
* 由于{{tr|projectiles}}扫过的面积很广，所以瞄准相对容易。再加上它的高{{tr|melee speed}}，玩家可使用它轻易的打败多体节的{{tr|bosses}}，如[[{{tr|Destroyer}}]]。
* 它极适用于[[{{tr|events}}]]，因为它具有强力的{{tr|attacks, projectiles}}和群控能力。

== 花絮 ==
* 与其余的[[{{tr|Old One's Army}}]]{{tr|weapons}}相同，{{tr|Flying Dragon}}直接参考了[[{{tr|Dungeon Defenders 2}}|《{{tr|Dungeon Defenders 2}}》]]中的同名{{tr|weapon}}，因为它是作为[[{{tr|crossover content}}|{{tr|crossover}}事件]]的一部分引入的。
* {{tr|Flying Dragon}}是游戏中最大的{{tr|sword}}之一，仅次于[[{{tr|Breaker Blade}}]]。
