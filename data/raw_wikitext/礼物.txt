'''{{tr|Presents}}'''是一种[[{{tr|grab bag}}]]游戏物品，有{{chance|1/13}}的几率掉落于[[{{tr|Christmas}}]]({{eicons|1.3.0.1|old-gen=y}} 12.15-12.31 / {{eicons|1.3.0.1|invert=y|old-gen=n}} 12.1-1.15)期间生成的任何[[{{tr|enemy}}]]（雕像生成的除外）。就像其他的摸彩袋一样，它们可以通过在玩家[[{{tr|inventory}}]]中单击{{key|open}}打开。{{tr|Presents}}可在任何时间，包括圣诞节之外的时间，打开。

{{eicons|1.3.0.1|old-gen=y}} {{tr|Presents}}在玩家的{{tr|inventory}}中时总是蓝色的，不过放置后则会是随机颜色的。

在{{eversions|1.2.2|invert=y}}，旧的[[File:Blue Present.png]]蓝色的、[[File:Green Present.png]]绿色的和[[File:Yellow Present.png]]黄色的{{tr|Presents }}仍然存在但无法放置。最初，由[[{{tr|Present Mimic}}]]掉落的{{tr|Presents}}可被放置，但在此之后会消失。它们不再掉落，但在{{3DS}}中尚未移除。若希望获取更多信息，参见[[{{tr|Present (furniture)}}]]。

== {{tr|Notes}} ==
* 任何最大{{tr|health}}多于 1 点，造成多于 0 点{{tr|damage}}的{{tr|enemy}}且掉落[[{{tr|coins}}]]的{{tr|enemies}}都有几率掉落{{tr|Present}}。{{source code ref|v=1.3.5.3|{{code|NPC.cs}}中的{{code|NPCLoot()}}函数。}} 如此一来，由[[{{tr|statues}}]]生成的{{tr|enemies}}、{{eil|Old One's Army}}和[[{{tr|Lunar Events}}]]中的{{tr|enemy}}和带翅膀的[[{{tr|Slimer}}]]永远都不会掉落{{tr|Presents}}。
* {{eicons|1.3.0.1}} {{tr|Presents}}可以像[[{{tr|furniture}}]]那样放置。每次放置时{{tr|Present}}的颜色都会随机改变。
* 若掉落的是{{tr|vanity set}}，则会一次性掉落三件。每一件都不会单独掉落。

== {{tr|Tips}} ==
* {{tr|Presents}}可以在[[{{tr|events}}]]期间相当容易的刷，比如[[{{tr|Solar Eclipse}}]]，其间会生成许多{{tr|enemies}}。

== {{tr|Trivia}} ==
* {{icon/mobile}} 在 1.1.1 {{tr|Christmas update}}后，{{tr|presents}}错误的掉落几乎所有物品。<ref>{{forums thread|best-thing-you-have-ever-got-from-presents.6680|Best thing you have ever got from presents}} {{small|Dec 8, 2014}}</ref> 该错误已被修复。
* 放置的{{tr|Present}}的其中一种外观和旧的{{tr|Green Present item}}颜色相同。
