'''{{tr|Beenade}}'''是一种[[{{tr|consumable ranged weapon}}]]，投掷时其行为类似于常规的[[{{tr|Grenade}}]]，但它会在 3 {{tr|seconds}}后的爆炸时释放 15 至 24 个友方{{tr|bee projectile}}。
这些{{tr|bee projectile}}每次击中造成{{tr|Beenade}} 100% 的{{tr|base damage}}（{{eicons|1.4.1}} {{iteminfo|Beenade|damage}} / {{eicons|1.4.1|invert=y}} 14 {{tr|base damage}}）{{source code ref|v=1.4.4.9|method=Terraria.Projectile.Kill}}，并且有 {{chance|1/2}} 的概率造成 1 点额外{{tr|damage}}。{{tr|Bee projectiles}}会持续追踪最近的[[{{tr|enemy}}]]，直到它 10 {{tr|seconds}}持续时间结束，穿透{{tr|enemy}}或于实心[[{{tr|block}}]]上反弹共三次，或者接触到[[{{tr|water}}]]或[[{{tr|lava}}]]而消失。{{tr|Beenade}}生成的友方{{tr|bee projectile}}不能造成[[{{tr|critical hit}}]]。

{{tr|Beenade}}有 {{chance|3/4}} / {{expert|100%}} 的概率掉落自[[{{tr|Queen Bee}}]]，一次性会掉落 10 至 30 个（{{expert|10 至 29 个}}），在 {{eversions|1.3.0.1}} 中，{{tr|Beenade}}也可以用一个[[{{tr|Grenade}}]]和一个[[{{tr|Bee Wax}}]]{{tr|craft}}。

== 备注 ==
* 装备{{eil|Hive Pack}}时，所有{{tr|Bee projectiles}}有 {{chance|1/2}} 的概率转化为{{tr|Giant Bee projectiles}}，它们会造成 1–3 点额外{{tr|damage}}，具有 {{kb|0.5}} [[{{tr|knockback}}]]，能穿透{{tr|enemy}}或于{{tr|solid block}}上反弹共 4 次，飞得更快，并且可持续 11 {{tr|seconds}}。两种{{tr|bee projectiles}}都会在{{tr|contact}}到[[{{tr|water}}]]或[[{{tr|lava}}]]后立即消失。{{source code ref|v=1.4.4.9|method=Terraria.Player.beeType / Terraria.Player.beeDamage / Terraria.Player.beeKB / Terraria.Projectile.SetDefaults|t=虽然{{tr|Giant Bees}}的实际{{tr|knockback}}为<code>0.5 + ''{{tr|Bee}}{{tr|knockback}}'' × 1.1</code>, 但是所有普通{{tr|Bees}}的{{tr|knockback}}都为 0，使 1.1 倍乘数相关计算永不生效。}}
* 由{{tr|Beenade}}造成的爆炸可受到{{tr|ranged damage}}加成、{{tr|ranged}}[[{{tr|critical strike}}]]加成以及来自[[{{tr|Frost armor}}]]的[[{{tr|Frostbite}}]]。然而，{{tr|Beenade}}生成的友方{{tr|bee projectile}}只能获得{{tr|ranged damage}}加成。它们不会获得[[{{tr|Frostbite}}]]而且无法造成{{tr|critical strike}}。{{source code ref|v=1.4.4.9|method=Terraria.Projectile.Kill / Terraria.Projectile.SetDefaults}}
* {{tr|Beenade}}产生的友方{{tr|bee projectile}}可以于[[{{tr|honey}}]]中生成，但浸没其中时它们的移动速度会慢许多。
* {{tr|Beenade}}在[[{{tr|water}}]]和[[{{tr|lava}}]]中爆炸时不会生成{{tr|bee projectile}}。
* 和所有{{tr|homing weapon}}一样，{{tr|Beenade}}生成的{{tr|bee projectile}}在 [[{{tr|PvP}}]] 中不能追踪其他{{tr|player}}。

== 小贴士 ==
* {{tr|Beenades}}较为适合在大型开阔空间中使用。因为在狭窄的空间中{{tr|bees}}更可能会于{{tr|block}}上反弹从而降低了潜在伤害，而其真正的伤害恰恰来自于蜂群。

== 花絮 ==
* “Beenade”（{{tr|Beenade}}）是“bee”（{{tr|bee}}）和“grenade”（{{tr|grenade}}）构成的混成词。
