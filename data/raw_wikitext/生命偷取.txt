'''{{tr|Lifesteal}}'''是能够以玩家造成{{tr|damage}}的一部分治疗玩家的效果，且可被{{eil|Moon Bite}}[[{{tr|debuff}}]]阻止。目前，有两种{{tr|items}}可以提供此效果：[[{{tr|Vampire Knives}}]]和[[{{tr|Spectre Hood}}]]。

== 细节 ==
{{tr|player}}是否能利用{{tr|lifesteal}}效果回复{{tr|health}}取决于 {{code|lifeSteal}} 变量。默认情况下，其最大值为 {{eicons|1.2.3}} 80 / {{expert|70}} / {{eicons|1.2.3|invert=y}} 120，每秒钟可以充能 {{eicons|1.2.3}} 36 / {{expert|30}} / {{eicons|1.2.3|invert=y}} 42 点{{source code ref|v=1.4.3.6|method=Terraria.Player.Update}}{{source code ref|v=1.2.2|old=y|method=Terraria.Player.UpdatePlayer}}。每当{{tr|player}}使用{{tr|lifesteal items}}造成{{tr|damage}}时，{{code|lifeSteal}} 就会减少，当 {{code|lifeSteal}} 小于等于 0 时，{{tr|the player}}就无法收到{{tr|lifesteal}}效果。如果造成的{{tr|damage}}足够高，那么 {{code|lifeSteal}} 就会降为负值，导致玩家在更长的一段时间内无法收到{{tr|lifesteal}}效果。例如，如果玩家在[[{{tr|Expert Mode}}]]中单次利用{{tr|lifesteal}}效果恢复了 100 {{tr|health}}，他们在 1 {{tr|second}}内就无法使用{{tr|lifesteal}}效果恢复{{tr|health}}。

进一步地，为了触发{{tr|lifesteal}}效果，目标的最大{{tr|health}}必须大于 5，且它的 {{code|canGhostHeal}} 参数必须被设置为 {{code|true}}（这是默认设置）。击中{{eil|Target Dummy}}也不会触发{{tr|lifesteal}}效果。{{source code ref|v=1.4.3.6|method=Terraria.Projectile.Damage}}

这些 {{tr|NPC}} 的 {{code|canGhostHeal}} 参数被设置为 {{code|false}}：{{source code ref|v=1.4.3.6|method=Terraria.NPC.SetDefaults}}

== {{tr|Notes}} ==
* 只有{{tr|projectile}}的{{tr|damage}}会触发{{tr|lifesteal}}效果。由{{tr|projectile}}造成的[[{{tr|debuff}}]]不会被考虑。
* 虽然行为与{{tr|lifesteal}}{{tr|weapon}}相似，但{{eil|Bat Bat}}在治疗时不消耗 {{code|lifeSteal}}，而是总会在击中时给{{tr|player}}治疗 1 {{tr|health}}。
* 与{{eil|Life Drain}}的名字和效果相悖的是，它提供的是[[{{tr|health regeneration}}]]而非{{tr|lifesteal}}。
