'''{{tr|Magic Missile}}'''是一个[[{{tr|pre-Hardmode}}]]的[[{{tr|wand}}]]。使用时，它会发射一枚{{tr|magic missile}}，按住 {{key|use}} 键时可随{{tr|cursor}}移动。若拖拽时松开 {{key|use}} 键，{{tr|missile}}会沿当前方向发射。若{{tr|cursor}}释放时固定不动，它会直线远离{{tr|player}}发射。释放后，{{tr|missile}}自动追踪附近的[[{{tr|enemies}}]]，当接触{{tr|enemy}}或[[{{tr|solid block}}]]时，它会产生小型爆炸。{{tr|Magic Missile}}有 {{chance|1/7}} 的概率从[[{{tr|Dungeon}}]]内[[{{tr|Gold Chest}}#{{tr|Dungeon}}|{{tr|Locked Gold Chests}}]]中找到，且有 {{chance|1/7}} 的概率从{{eil|Golden Lock Box}}中获得。

== 备注 ==
* [[{{tr|mana}}]]{{tr|cost}}仅用于初始施法。维持和导向{{tr|the missile}}没有额外的{{tr|mana cost}}，玩家在控制它时就会再生{{tr|mana}}。
* {{tr|The Magic Missile}}的{{tr|projectile}}击中时造成 8 {{tr|ticks}}{{tick}}[[{{tr|global invincibility frame}}]]。{{source code ref|v=1.4.4.9|method=Terraria.Projectile.Damage}}
* 释放之后，{{tr|projectile}}可追踪 50 {{tr|tiles}}半径内的目标。{{source code ref|name=magicmissiles|v=1.4.4.9|method=Terraria.Projectile.AI_009_MagicMissiles}}
* 当{{tr|the projectile}}被破坏时，其会产生以碰撞点为中心的 8×8 {{tr|tile}}方形范围的爆炸。{{source code ref|v=1.4.4.9|method=Terraria.Projectile.Kill}}
* {{tr|The Magic Missile}}可以穿过[[{{tr|water}}]]和[[{{tr|honey}}]]发射（不像另一个武器[[{{tr|Flamelash}}]]）并且也不会被{{tr|liquids}}减速。{{source code ref|v=1.4.4.9|method=Terraria.Projectile.SetDefaults}}

== 小贴士 ==
* {{tr|Magic Missile}}的{{tr|The projectiles}}可以受控穿过一格宽的缝隙。
* 因为{{tr|the missile}}会发光且可控，此{{tr|weapon}}是不错的探索工具。

== 花絮 ==
* {{tr|Magic Missile}}{{tr|spell}}最早在 1974 年出现于奇幻角色扮演桌游《''{{wikipedia|lang=en|Dungeons & Dragons|龙与地下城（Dungeons & Dragons）}}''》中，之后出现在无数的出版物中。
** 在《''龙与地下城（Dungeons & Dragons）''》中，{{tr|the magic missile}}{{tr|spell}}总是会击中目标，在《''[[{{tr|Terraria}}]]''》则用{{tr|weapon}}的追踪能力表示。
* 以前，在{{mobile version}}上，{{tr|the missile}}可以发射多次，叠加成可以秒杀大多数{{tr|enemies}}的{{tr|projectile}}。
