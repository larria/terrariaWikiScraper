'''{{tr|Shimmer}}'''是一种有光泽的淡紫色珠光[[{{tr|liquid}}]]。在游戏的任何时期中，都可以在[[{{tr|Aether}}]]中找到它，后者是一个仅会在每个{{tr|world}}中的[[{{tr|Underground}}]]或[[{{tr|Cavern}}]][[{{tr|layer}}]]中生成一个的[[{{tr|mini biome}}]]，且总与[[{{tr|Jungle}}]]生成在世界的同一侧。在击败{{tr|Moon Lord}}后，可以通过使用[[{{tr|Bottomless Shimmer Bucket}}]]来无限地获得它。

{{tr|The Shimmer}}有着转化（“嬗变”）浸入其中的{{tr|items}}和生物的特殊能力，这包括：
* 大部分制作好的{{tr|items}}会被转化回（“{{tr|decraft}}”为）其{{tr|crafting materials}}。
* 特定的{{tr|items}}会被转化为其他{{tr|item}}。某些物品只能通过这种转化获得，如[[{{tr|Ambrosia}}]]。某些{{tr|item}}的转化只能特定条件下实现。
* {{tr|town NPC}} 的{{tr|sprite}}会被改变。
* 特定种类的{{tr|enemies}}会被转化为其他{{tr|enemy}}。
* 大部分[[{{tr|Critters}}]]会被转化为[[{{tr|Faelings}}]]。
被转化后的{{tr|Items}}会向上浮起并离开{{tr|Shimmer}}，在被玩家拾取前会悬浮在半空并发出明亮的光，类似于[[{{tr|souls}}]]。任何无法被{{tr|Shimmer}}转化的物品都会遵循正常[[{{tr|gravity}}]]并沉到{{tr|Shimmer}}底部。

除了其转换效果，{{tr|Shimmer}}对于进入它的其他事物也有效果：
* 当玩家的头部浸没在{{tr|liquid}}中，或玩家的[[{{tr|fishing pole}}]]的{{tr|bobber}}碰到{{tr|liquid}}时，他会被施加[[{{tr|Shimmering}}]]{{tr|debuff}}，开始缓慢下降并穿过所有{{tr|blocks}}，直到到达开阔处。
* 特定的[[{{tr|projectiles}}]]会在接触{{tr|Shimmer}}池时向上弹起（如下落的[[{{tr|Fallen Star}}]]）。
* 当没有特殊嬗变，且不免疫[[{{tr|Shimmering}}]]{{tr|debuff}}的{{tr|enemies}}（例：[[{{tr|Werewolf}}]]）进入{{tr|Shimmer}}时，它们会先逐渐变得透明，直到完全隐形。然后它们会缓慢变得越来越不透明。攻击{{tr|enemies}}会让其更快变得不透明。{{source code ref|v=1.4.4.9|method=Terraria.NPC.UpdateNPC_BuffApplyVFX}}

和其他{{tr|liquids}}不同，玩家无法在{{tr|Shimmer}}中[[{{tr|fishing}}]]或使用物品，也无法用[[{{tr|Empty Bucket}}]]盛起它。{{tr|Shimmer}}也没有相对应的[[{{tr|Liquid bombs}}|{{tr|liquid bomb}}]]{{tr|variant}}。不过，它仍然能被[[{{tr|pumps}}]]移动。

在和任何其他[[{{tr|liquid}}]]接触时，{{tr|Shimmer}}会形成[[{{tr|Aetherium Block}}]]。

== {{tr|decraft}}特例 ==
除了普通的{{tr|decraft}}以外，许多{{tr|recipes}}拥有特殊的{{tr|decraft}}结果和规则，在此 wiki 上称为“{{tr|decraft}}特例”。{{tr|decraft}}特例包括：{{source code ref|v=1.4.4.9|method=Terraria.Recipe.SetupRecipes}}
* 有多种{{tr|recipes}}的{{tr|item}}只会被{{tr|decraft}}为特定的一套{{tr|ingredients}}。<!--Technically, these items have multiple recipes registered, with some of the recipes being marked as "notDecraftable".{{source code ref|v=1.4.4.9|field=Terraria.Recipe.notDecraftable}}-->
** 例如，[[{{tr|Frost armor}}]]可用[[{{tr|Adamantite Bar}}]]或[[{{tr|Titanium Bar}}]]制作，但它总是会被{{tr|decraft}}为{{tr|Adamantite Bars}}，即使是在有{{tr|Titanium}}的{{tr|worlds}}中也如此。
* 拥有两种[[{{tr|evil biome}}]]{{tr|recipes}}的某些{{tr|items}}会基于当前{{tr|world}}的{{tr|evil biome}}被{{tr|decraft}}。{{source code ref|v=1.4.4.9|method=Terraria.Recipe.UpdateWhichItemsAreCrafted/Terraria.GameContent.ShimmerTransforms.GetDecraftingRecipeIndex|field=Terraria.Recipe.corruption/Terraria.Recipe.crimson}}
** 例如，一个[[{{tr|Monster Lasagna}}]]在[[{{tr|Corruption}}]]{{tr|world}}中总是会被{{tr|decraft}}为 8 个[[{{tr|Rotten Chunk}}]]，在[[{{tr|Crimson}}]]{{tr|world}}中则是 8 个[[{{tr|Vertebra}}]]。此外，在 [[{{tr|drunk world}}]] 中，{{tr|decraft}}结果遵循当天的{{tr|evil biome}}。
* 一些{{tr|items}}会被{{tr|decraft}}为独特的一套{{tr|item}}。{{source code ref|v=1.4.4.9|method=Terraria.Recipe.AddCustomShimmerResult}}
** 例如，[[{{tr|Copper Pickaxe}}]]会被{{tr|decraft}}为[[{{tr|Wood}}]]和[[{{tr|Copper Ore}}]]，即使{{tr|Copper Ore}}不是其{{tr|crafting recipe}}的一部分。
* 在[[{{tr|Placed Bottle}}]]或[[{{tr|Alchemy Table}}]]制作的{{tr|Items}}会被{{tr|decraft}}为其{{tr|ingredients}}的随机一部分。{{source code ref|v=1.4.4.9|method=Terraria.Item.GetShimmered/Terraria.Recipe.AddRecipe}}
** 这一随机性是通过让{{tr|decraft}}结果中的每个单独的{{tr|item}}有 {{chance|1/3}} 几率消失来达成的。<ref>例如，当{{tr|decraft}} 1 个[[{{tr|Shine Potion}}]]时，[[{{tr|Bottled Water}}]]、[[{{tr|Daybloom}}]]、[[{{tr|Glowing Mushroom}}]]均有 {{chance|1/3}} 几率消失；{{tr|decraft}} 9999 个{{tr|Shine Potions}}时，{{tr|decraft}}结果包括一叠 9999 个{{tr|Dayblooms}}。不过，在这一叠{{tr|item}}被实际创建前，其中的每一个{{tr|Daybloom}}都有 {{chance|1/3}} 几率消失。</ref>因此，平均来说{{tr|decraft}}在{{tr|Placed Bottle or Alchemy Table}}制作的{{tr|items}}仅能获得 {{chance|2/3}} 的{{tr|ingredients}}。
* 用[[{{tr|Bone}}]]制作的{{tr|Items}}在击败[[{{tr|Skeletron}}]]前无法被{{tr|decraft}}。{{source code ref|name=updateRecipeSets|v=1.4.4.9|method=Terraria.GameContent.ShimmerTransforms.UpdateRecipeSets}}
* 用[[{{tr|Lihzahrd Bricks}}]]制作的{{tr|Items}}在击败[[{{tr|Golem}}]]前无法被{{tr|decraft}}。<ref name="updateRecipeSets"/>
* 某些{{tr|items}}无法被{{tr|decraft}}，以防止不正当利用或不好的含义。{{source code ref|v=1.4.4.9|field=Terraria.Recipe.notDecraftable}}
** 例如，{{tr|decraft}}[[{{tr|Blizzard in a Bottle}}]]（可以在[[{{tr|Ice biome}}]]中的[[{{tr|Chests}}]]内找到）会允许在[[{{tr|pre-Hardmode}}]]获得[[{{tr|Frost Core}}]]，{{tr|decraft}}[[{{tr|Bunny Stew}}]]则意味着让死掉的、已被烹饪的[[{{tr|Bunny}}]]重新活过来。

== 备注 ==
* 在接触{{tr|Shimmer}}时会被反弹的{{tr|Projectiles}}仅会在它们接触{{tr|Shimmer}}池的上表面时被反弹。把它们射向{{tr|Shimmer}}池的侧面或底面（可通过使用[[{{tr|Bubble}}]]实现）会立即摧毁{{tr|projectile}}。
* 由{{tr|Statue}}生成的生物不会在{{tr|Shimmer}}中被转化，而是会消失。
* 被{{tr|Shimmer}}变得隐形的{{tr|enemies}}可以用[[{{tr|Hunter Potion}}]]看到。
* 虽然本身是用在[[{{tr|water}}]]上的，[[{{tr|Super Absorbant Sponge}}]]也能吸收{{tr|Shimmer}}，而[[{{tr|Liquid Sensor (Water)}}]]也能探测{{tr|Shimmer}}。
* 当{{tr|crafting recipe}}会产生一叠{{tr|items}}的{{tr|item}}（如[[{{tr|Crystal Bullets}}]]）被{{tr|decraft}}时，需要向{{tr|Shimmer}}中投入相同数量的该物品才能进行{{tr|decraft}}。多余的物品不会被转化。
* 由{{tr|decraft}}产生的[[{{tr|Fallen Star}}]]不会在{{tr|daytime}}消失。然而，如果玩家拾取它们失败了（例：如果它们被吸向玩家，但最终没有被玩家拾取），它们仍然会消失。
* 当装备了[[{{tr|Chromatic Cloak}}]]来避免[[{{tr|Shimmering}}]]时，浸没于{{tr|Shimmer}}和浸没于[[{{tr|water}}]]中相当：玩家的水平和垂直[[{{tr|movement speed}}]]会[[{{tr|Movement speed|link=y}}#{{tr|Medium movement}}|变慢]]相同的量，可以用诸如[[{{tr|Flippers}}]]的{{tr|accessories}} 在{{tr|the Shimmer}}中[[{{tr|Swimming}}]]，且玩家装备[[{{tr|Neptune's Shell}}]]{{tr|or its upgrades}}时会变形为[[{{tr|Merfolk}}]]。不过，玩家不会在{{tr|Shimmer}}中[[{{tr|drown}}]]；不会出现[[{{tr|breath meter}}]]。
* 在[[{{tr|Expert Mode}}]]中，在[[{{tr|Snow biome}}]]或[[{{tr|Ice biome}}]]中接触{{tr|Shimmer}}会对玩家施加[[{{tr|Chilled}}]][[{{tr|debuff}}]]。

== 小贴士 ==
* 可以用多种方式来利用{{tr|Shimmer}}的{{tr|decraft}}特性：
** 可以通过重复制作和{{tr|decraft}}装备来为装备刷出一个好的[[{{tr|modifiers}}]]。
** {{tr|decraft}}多余的或较差的装备可取回{{tr|materials}}。
** 在{{tr|world}}各处找到的{{tr|Items}}都可被拿来{{tr|decraft}}，包括[[{{tr|Underground Cabin}}]]的{{tr|furniture}}和[[{{tr|Magic Mirror}}]]。
** 在[[{{tr|Multiplayer}}]]中，可以{{tr|decraft}}多余的[[{{tr|Recall Potion}}]]，可能获得[[{{tr|Specular Fish}}]]。{{tr|Specular Fish}}可以被制作为[[{{tr|Wormhole Potion}}]]。
*** 类似地，在{{tr|singleplayer}}中，任何得到的{{tr|Wormhole Potions}}都可被{{tr|decraft}}，可能获得{{tr|Specular Fish}}。
** 从 [[{{tr|NPCs}}]] 处购买{{tr|Items}}，然后将其{{tr|decraft}}可获得资源。
*** 可在[[{{tr|Merchant}}]]处以 {{iteminfo|Iron Anvil|buy}} 的价格购买[[{{tr|Iron Anvil}}]]，然后{{tr|decraft}}为五个[[{{tr|Iron Bar}}]]。
*** 可在[[{{tr|Jungle}}]]中与{{tr|Merchant}}对话，以 {{iteminfo|Furnace|buy}} 的价格购买[[{{tr|Furnace}}]]，然后{{tr|decraft}}为 20 个[[{{tr|Stone Block}}]]、4 个[[{{tr|Wood}}]]、和 3 个[[{{tr|Torch}}]]。
