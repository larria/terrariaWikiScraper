'''{{tr|Granite Golem}}'''是一个[[{{tr|pre-Hardmode}}]][[{{tr|enemy}}]]，生成于[[{{tr|Granite Cave}}]]。每 5 {{tr|seconds}}，它会进入防御状态，令所有{{tr|attacks}}无效，使自身完全无敌 2 {{tr|seconds}}。{{source code ref|name=ai|v=1.4.3.6|method=Terraria.NPC.AI_003_Fighters}}在此状态下，它仍然可以对玩家造成伤害。

和[[{{tr|Granite Elemental}}]]一样，它总会掉落少量[[{{tr|Granite Block}}]]。除此之外，它会以低概率掉落[[{{tr|Night Vision Helmet}}]]。

== {{tr|Notes}} ==
* {{tr|Granite Golem}}每 5 {{tr|seconds}}进入一次无敌状态，每次命中将会减少 0.25 {{tr|seconds}}的冷却时间。它只有站在任何一种表面上时才能进入无敌状态。<ref name="ai"/>

== {{tr|Tips}} ==
* {{tr|player}}可以通过{{tr|Granite Golem}}的叫声来判断附近存在{{tr|Granite Cave}}。
* {{tr|Granite Golems}}可以通过使用[[{{tr|Wire}}]]激活[[{{tr|Granite Golem Statue}}]]的方式召唤。这是一个十分有效的用来刷取[[{{tr|Geode}}]]（用以获取[[{{tr|gems}}]]）和[[{{tr|Night Vision Helmet}}]]的方法。其中，{{tr|Night Vision Helmet}}可以以{{iteminfo|Night Vision Helmet|sell}}的价值售出。

== {{tr|Trivia}} ==
* 直到[[1.3.5]]，{{tr|enemies}}都有一个未使用的内部名称。{{tr|Granite Golem}}的名称是''Grolem''{{source code ref|v=1.3.4.4|old=y|method=Terraria.NPC.SetDefaults}}，一个''granite（花岗岩）''和''golem（魔像）''的{{wikipedia|portmanteau|合成词}}，与[[{{tr|Granite Elemental}}]]的''Grelimental''类似。
* 在犹太传说中，{{wikipedia|golem|魔像}} ({{ipa|ˈɡoʊləm|before=}}, ''GOH-ləm''; 希伯来语： גולם‎) 是完全由无生命物质创造出的拟人生命(通常为泥土和泥浆)。在圣诗和中世纪的创作中，这个词被用来代指一种无定形或者未成形的材料。
