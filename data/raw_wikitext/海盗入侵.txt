'''{{tr|Pirate Invasion}}'''是一个[[{{tr|Hardmode}}]][[{{tr|invasion}}]][[{{tr|event}}]]，会使得{{tr|pirate}}主题的{{tr|enemies}}代替常规{{tr|enemies}}生成。在满足特定条件后，{{tr|Pirate Invasion}}可能会在任何[[{{tr|dawn}}]]时随机发生。它也可以用[[{{tr|Pirate Map}}]]手工召唤。

在{{tr|event}}开始后，{{tr|Pirate Invasion}}会从{{tr|world}}的随机一侧始发（由下列两个[[{{tr|status messages}}]]之一提示：''“{{gameText|LegacyMisc.25}}”''/''“{{gameText|LegacyMisc.26}}”''）并向[[{{tr|world spawn point}}]]移动。{{tr|status message}}''“{{gameText|LegacyMisc.27}}”''会在{{tr|Pirate Invasion}}到达了{{tr|world spawn point}}时显示。{{tr|Event enemies}}仅能在{{tr|Pirate Invasion}}的位置附近或在{{tr|town}} [[{{tr|NPCs}}]] 附近生成。''见[[#{{tr|Notes}}|{{tr|Notes}}]]获取细节。''

{{tr|Pirate Invasion}}在击杀足够多来自它的{{tr|enemies}}后即完成。在{{eversions|1.3.0.1}}中，会显示进度条来展示{{tr|event}}进度。

击败{{tr|Pirate Invasion}}至少一次会满足[[{{tr|Pirate}}]] [[{{tr|NPCs}}|{{tr|NPC}}]] 入住的条件。

== 召唤 ==
如果{{tr|world}}中至少有一名[[{{tr|player}}]]拥有至少 200 最大[[{{tr|health}}]]，就可以用[[{{tr|Pirate Map}}]]来手工召唤{{tr|Pirate Invasion}}。{{source code ref|v=1.4.4.9|method=Terraria.Player.ItemCheck_UseEventItems}}如果没有玩家满足此条件或已经有{{tr|invasion event}}激活，{{tr|Pirate Map}}将没有效果且不会被消耗。
* 尽管{{tr|Pirate Maps}}只能在[[{{tr|Hardmode}}]]期间获得，它可以在[[{{tr|pre-hardmode}}]]的{{tr|worlds}}中使用。

如果满足了以下条件，{{tr|Pirate Invasion}}在{{tr|dawn}}（{{time|4:30 AM}}）也会（以一定几率）随机发生：{{source code ref|v=1.4.4.9|method=Terraria.Main.UpdateTime_StartDay}}
* 没有其他[[{{tr|invasion}}]]{{tr|events}}或[[{{tr|Old One's Army}}]]激活。
* [[{{tr|Goblin Army}}]]{{tr|event}}未能发生。
* {{tr|world}}处于[[{{tr|Hardmode}}]]。
* 至少摧毁了一个[[{{tr|Altar}}]]。
* {{tr|world}}中至少有一名{{tr|player}}拥有至少 200 最大{{tr|health}}。

假设{{tr|Goblin Army}}已被打败一次，{{tr|Pirate Invasion}}发生的具体几率如下：
* 首先，[[{{tr|Goblin Army}}]]{{tr|event}}必须未能发生（{{chance|59/60}} 几率）。
* 若尚未打败{{tr|Pirate Invasion}}，几率为 {{chance|1/30}}。
* 若至少打败{{tr|Pirate Invasion}}一次，几率为 {{chance|1/60}}。
* 每个{{tr|dawn}}的最终几率是首次{{tr|Pirate Invasion}}为 {{percent|{{#expr:59/60*1/30}}}}，后继{{tr|invasions}}为 {{percent|{{#expr:59/60*1/60}}}}。

如果玩家不使用[[{{tr|bed}}]]或[[{{tr|Enchanted Sundial}}]]加速时间，平均来说每游玩 20.4 小时会随机发生 1 次{{tr|Pirate Invasion}}（条件满足后的首次{{tr|invasions}}则为 12.2 小时）。

== 完成 ==
{{tr|Pirate Invasion event}}在击杀足够多来自此{{tr|event}}的{{tr|enemies}}后即完成。{{tr|The status message}}''“{{gameText|LegacyMisc.24}}”''会在{{tr|event}}完成时显示。

所需击杀数和{{tr|event}}开始时{{tr|world}}中具有至少 200 {{tr|maximum health}}的玩家数量成正比，可以通过下方公式计算：
: <math>\text{击杀数}=120+60\times\text{玩家数量}</math>

例如：
* 在{{tr|singleplayer}}中，所需击杀数为 <math>120+60\times1=180</math>。
* 在[[{{tr|multiplayer}}]]中有 8 个满足条件的玩家时，所需击杀数为 <math>120+60\times8=600</math>。
* 一次{{tr|invasion}}的最高击杀数为 {{formatnum:{{#expr:120+60*255}}}}（255 个玩家）。

大多数{{tr|enemies}}算作 1 次击杀，但某些{{tr|enemies}}可能算作更多或更少。{{source code ref|v=1.4.4.9|method=Terraria.NPC.checkDead}}详细见下表：{{options/set|item|mode=table|maxsize=50px}}

== 备注 ==
* {{tr|Pirate Invasion}}和其他[[{{tr|invasion}}]]{{tr|events}}一样拥有下列特性：
** 提示{{tr|invasion}}开始的{{tr|status message}}（''“{{gameText|LegacyMisc.25}}”''/''“{{gameText|LegacyMisc.26}}”''）在[[{{tr|world size}}|{{tr|small or medium}}]]{{tr|worlds}}内显示一次，在[[{{tr|world size}}|{{tr|large}}]]{{tr|worlds}}内显示两次。
** [[{{tr|Death}}]]或退出{{tr|world}}不会结束{{tr|invasion}}，也不会影响{{tr|invasion}}的进度。
** 当前{{tr|invasion}}的类型、进度、和所需击杀数会在退出{{tr|world}}时保存。
*** 因此，当{{tr|invasion}}发生时没有阻止它的方法。
** {{tr|Pirate Invasion}}激活时，{{tr|Town}} [[{{tr|NPCs}}]] 无法{{tr|respawn}}。
** {{tr|invasions}}开始于左侧或右侧[[世界边界]]并以每{{tr|tick}}{{tick}} 1 格的速度移动到{{tr|world}}[[{{tr|spawn point}}]]处（或者说每秒 60 格；{{eicons|1.3.0.1}}此速度受时间速度影响，例如：[[{{tr|Journey Mode}}]]时间滑条、[[{{tr|Enchanted Sundial}}]]、[[入睡]]）。{{source code ref|v=1.4.4.9|method=Terraria.Main.UpdateInvasion}}假设{{tr|world spawn point}}位于{{tr|world}}的水平中心，{{tr|invasions}}在{{tr|small, medium, and large}}[[{{tr|world size}}]]内分别需要 {{#expr:4200/2/60 round 0}}、{{#expr:6400/2/60 round 0}}、和 {{#expr:8400/2/60 round 0}} 秒才能到达{{tr|spawn point}}。
** 玩家必须不低于{{tr|surface}}以下 67.5 格且位于{{tr|invasion}}位置的水平方向（两侧）187.5 格以内才能生成{{tr|invasion enemies}}。此种情况下，来自{{tr|Pirate Invasion}}的{{tr|enemies}}会完全覆盖当前环境中其他的[[{{tr|NPC spawning}}|{{tr|enemy}}生成]]。{{source code ref|v=1.4.4.9|method=Terraria.NPC.SpawnNPC}}
*** 如果{{tr|invasion}}到达了{{tr|world}}水平中心的 5 格以内的位置，则当玩家位于任何{{tr|town NPC}} 水平方向（两侧）187.5 格以内时也可以通过水平范围检测。{{tr|The town NPC}} 的垂直位置不会被检测。此种情况下，来自{{tr|Pirate Invasion}}的{{tr|enemies}}只有 {{chance|2/3}} 几率覆盖其他{{tr|enemy}}生成。
** {{tr|An invasion}}不会阻止大多数其他{{tr|events}}发生（如[[{{tr|Blood Moon}}]]、[[{{tr|Solar Eclipse}}]]、或{{tr|night}}的随机 [[{{tr|Boss}}]] 生成）。不过，非 {{tr|boss enemy}}的生成仍然可能被{{tr|invasion enemy}}生成覆盖。
* {{eicons|Flying Dutchman}}如果在有来自另一次{{tr|invasion}}的{{tr|Flying Dutchman}}依然存活的情况下开始{{tr|Pirate Invasion}}，所有在使用了{{tr|pirate map}}之后被杀掉的由那艘{{tr|Flying Dutchman}}生成的{{tr|Pirates}}都会计入{{tr|invasion}}进度中。

== 小贴士 ==
{{strategies|t={{tr|Pirate Invasions}}|指南:{{tr|Pirate Invasion}}攻略}}
* {{tr|The Pirate Invasion}}类似于[[{{tr|Goblin Army}}]]，但{{tr|the pirates}}整体上更强，他们没有像[[{{tr|Goblin Sorcerer}}]]那样的{{tr|caster}}，也没有和[[{{tr|Goblin Summoner}}]]相对应的{{tr|enemy}}。鉴于这些弱点，大部分相同的策略和准备工作用在{{tr|The Pirate Invasion}}中会更加有效。
* 由于{{tr|Pirate enemies}}没有穿墙攻击，玩家可以用{{tr|blocks}}筑墙遮蔽自身，然后在其中安全地进行攻击。例如，使用{{eil|Daedalus Stormbow}}（理想情况下用[[{{tr|Holy Arrow}}]]来最大化伤害），或者在墙上挖个小洞并使用{{eil|Yoyos|yoyo}}、[[{{tr|Magic Dagger}}]]、{{eil|Flying Knife}}、或其他{{tr|projectile weapon}}进行攻击。
** {{eicons|1.4.0.1}}不过，玩家应当小心击杀[[{{tr|Pirate Captain}}]]所生成的[[{{tr|Pirate's Curse}}]]，它们能穿墙移动并对玩家造成高额伤害。
* 如果使用了[[{{tr|Flask of Gold}}]]的情况下进行战斗，每次{{tr|event}}平均可以收入 45–{{gc|55}}，从而使其成为最简单的[[指南:赚钱|赚钱]]方法之一。
* 若安排在相应的[[{{tr|biome}}]]中与{{tr|the Pirates}}作战，{{tr|Pirate Invasions}}也可用于刷[[{{tr|Biome Key}}]]。一种方式是在{{tr|spawn point}}或{{tr|town}} [[{{tr|NPC}}]]附近设置人工{{tr|biome}}。
* 若在[[{{tr|Ocean}}]]{{tr|biome}}中战斗，{{tr|Pirates}}自己也可以掉落[[{{tr|Pirate Map}}]]，这可以让刷取变得更简单。其他[[{{tr|invasion}}]]也可以用来刷{{tr|Pirate Maps}}。
* 由于{{eil|Flying Dutchman}}会生成{{tr|pirates}}，有可能将{{tr|the Flying Dutchman}}停下甚至困住并只击杀其生成的{{tr|enemies}}。
** 将{{tr|Flying Dutchman}}困住可以创造出稳定的{{tr|enemies}}来源，这可以被用来无限地刷钱，例如使用[[{{tr|Slime Staff}}]]和[[{{tr|Lucky Coin}}]]。

== 花絮 ==
* {{tr|The Pirate invasion}}的{{tr|enemies}}是[[《{{tr|Terraria}}》]]中少数可确定是人类的{{tr|enemies}}。
* 当{{tr|Pirate Deckhand}}和{{tr|Pirate Corsair}}被迫通过 2 格高的空间时，他们会以手臂放下的状态站住几秒钟，然后开始远离玩家。
* 尽管{{tr|achievement}}描述说{{tr|pirates}}会抢走玩家的战利品，{{tr|event}}中的{{tr|enemy}}并不能带走玩家的任何{{tr|items}}。
* 基础的{{tr|Pirate Invasion}}{{tr|enemy}}（不包括[[{{tr|Parrot}}]]、[[{{tr|Pirate Captain}}]]、和[[{{tr|Flying Dutchman}}]]）目前为止拥有最多数量的可能{{tr|drops}}，有 33 种不同的{{tr|item}}。
