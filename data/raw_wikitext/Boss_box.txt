Template for displaying a boss box; uses {{tl|infocard}}.

== Parameters ==
; name
Name of the boss

; notrophy
Suppress the trophy in front of the strategies link

; trophy
Custom trophy, should be used like this: <code>[[File:''&lt;trophy&gt;''.png|link=''&lt;trophy&gt;'']]</code>

; strategies
Custom link after the "''... recommendations, see ''"

; color
Background color of the image

; animated
Displayed image is <code>Animated <nowiki>{{{name}}}</nowiki>.gif</code> by default. Use <code>animated=n</code> to change it to <code><nowiki>{{{name}}}</nowiki>.png</code>

; imagealt
Completely customized image

; summoned
Text beneath the image

; 1st unnamed parameter
Main description text
