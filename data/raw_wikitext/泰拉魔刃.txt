'''{{tr|Terragrim}}'''是一个罕见的[[{{tr|pre-hardmode}}]]的[[{{tr|melee weapon}}]]，外形看起来是一把[[{{tr|sword}}]]，但它在使用时并不会显现出实体剑刃。{{tr|Terragrim}}在[[{{tr|autoswing}}]]时既不是[[{{tr|broadsword}}]]那样的从头顶挥舞过去，也不是[[{{tr|shortsword}}]]那样的刺击，而是朝向光标方向的一系列难以分辨的劈砍。此区域内的任何东西都会以远超过该{{tr|weapon}}所述的[[{{tr|use time}}]]的频率受到持续{{tr|damage}}。这个{{tr|damage}}区域延伸到相当大的一片范围，甚至能够攻击到{{tr|player}}背后的一小段距离处。

{{float file box|width=188px|title=石中剑背景物体|[[File:Enchanted Sword comparison.png]]|垃圾（左）和石中{{tr|Enchanted Sword}}（右）。只有右侧的那种有几率掉落{{tr|Terragrim}}。}}

可以通过摧毁在[[{{tr|Enchanted Sword Shrine}}]]中或非常罕见的情况下在{{tr|Caverns layer}}中出现的独特的[[{{tr|background object}}]]来获得{{tr|Terragrim}}。该物体被摧毁时，有 {{chance|1/50}} 几率掉落{{tr|Terragrim}}，其余时候则掉落[[{{tr|Enchanted Sword}}]]。

[[{{tr|Arkhalis}}]]是{{tr|Terragrim}}的[[{{tr|Hardmode}}]]、[[{{tr|Expert Mode}}]]独有的升级物。

{{modifier|meleealt}}{{tr|The Terragrim}}的独特攻击方式不受任何大小和速度{{tr|modifiers}}影响。

== 备注 ==
* {{tr|The Terragrim}}击中时产生 5 [[{{tr|global invincibility frame}}]]，{{source code ref|v=1.4.4.9|method=Terraria.Projectile.Damage}}这意味着它可以每秒击中目标 12 次。
* 在 [[{{tr|Celebrationmk10}}]] 和 [[{{tr|Get fixed boi}}]] {{tr|worlds}}中，{{tr|the Terragrim}}可以在[[{{tr|Blood Moon}}]]期间于[[{{tr|Princess}}]]处以 {{iteminfo|Terragrim|buy}} 的价格购买。
* 虽然外观相同，[[{{tr|Rubblemaker}}]]放置的{{tr|Enchanted Swords}}被摧毁时永远不会掉落{{tr|Terragrim}}。{{source code ref|v=1.4.4.9|method=Terraria.WorldGen.Check3x2}} <!-- different tile ids -->

== 小贴士 ==
* 其高{{tr|attack speed}}使得在与第二阶段的[[{{tr|Brain of Cthulhu}}]]战斗时，玩家即使站定不动也可以不受伤害，因此用它很容易击败此 {{tr|boss}}。
* 寻找{{tr|Terragrim}}的最佳[[{{tr|world size}}]]为{{tr|medium world}}。虽然在{{tr|large world}}通常有更多的{{tr|Enchanted Sword Shrine}}（2–4 次生成尝试，{{tr|medium world}}中只有 1–3 次），但在{{tr|medium world}}中，它们的平均距离通常更短。
* 由于{{tr|Terragrim}}是否掉落是在{{tr|background object}}被破坏时才确定的，可以通过反复读档来迫使其掉落：找到一个带有石中{{tr|The Enchanted Sword}}的{{tr|Enchanted Sword Shrine}}，关闭[[{{tr|Autosave On}}|{{tr|autosave}}]]，然后重复破坏{{tr|background object}}和强制退出[[《{{tr|Terraria}}》]]（''不是''通过暂停菜单中的"{{gameText|LegacyInterface.35}}"，而是关闭《{{tr|Terraria}}》应用程序本身，例如{{icon/windows}} Windows 用户可以按下{{key|Alt}}+{{key|F4}}）直到获得{{tr|Terragrim}}而不是{{tr|Enchanted Sword}}。虽然这种方法可能被许多人认为是作弊，但是它对于一些{{tr|players}}来说很有用。

== 花絮 ==
* {{tr|The Terragrim}}的名称参考了名为 [https://castlevania.fandom.com/wiki/Crissaegrim Crissaegrim] 的剑，后者也被称为 Valmanway，来自 1997 年的动作角色扮演平台游戏{{维基百科|b=y|Castlevania: Symphony of the Night|t=恶魔城：月下夜想曲}}。和[[{{tr|Arkhalis}}]]一样，{{tr|Terragrim}}的功能和外观也是基于 Crissaegrim 的。
* 尽管{{tr|Terragrim}}出现在了[[{{tr|Zenith}}]]的攻击动画中，它并不是制作{{tr|Zenith}}所需的{{tr|crafting material}}。
* 由于{{tr|Arkhalis}}在 [[1.4.0.1]] 中被包含在[[{{tr|Arkhalis's set}}|Arkhalis 的{{tr|developer set}}]]中，{{tr|The Terragrim}}替代了{{tr|Arkhalis}}作为在{{tr|Enchanted Sword Shrines}}中找到的特殊{{tr|weapon}}。
* 即使使用[[{{tr|inventory editor}}]]来获得{{tr|Legendary modifier}}，它也不会改变{{tr|Terragrim}}劈砍的速度或大小。
* {{tr|The Terragrim}}是少数可以作为游戏{{tr|world}}中的实际物体获得的{{tr|weapons}}之一，其余的为[[{{tr|Enchanted Sword}}]]、[[{{tr|Water Bolt}}]]、和[[{{tr|Abigail's Flower}}]]。
