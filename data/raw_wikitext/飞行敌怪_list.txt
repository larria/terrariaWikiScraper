List of all flying enemies. It is generated via [[DPL]], specifically the two categories [[:Category:Flying Enemy NPCs]] and [[:Category:Hovering AI NPCs]].


Optional parameters:

; width
: = width of item; default is 18em

; hardmode
: = no; only displays pre-Hardmode enemies
: = yes; only displays Hardmode enemies
: defaults to displaying both

The filename and size of the images are controlled by the code itself. They can be changed if needed.
