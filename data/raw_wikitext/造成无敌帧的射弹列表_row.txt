== Parameters==
; id or first unnamed parameter
The id of the projectile. Projectile info is automatically retrieved with presented id with {{tl|getProjectileInfo}}.

; size
Resize the image of projectile (<code>size</code>px).

; type
The type of [[invincibility frame]] the projectile applies. Register localization info for invincibility type on this page.

; length
The length of invincibility frame applied on hit. Automatically determined and calculated if invincibility frame type is static or local. 
* For static: equals to <code>idStaticNPCHitCooldown</code>
* For local: calculated with <code>localNPCHitCooldown</code> and <code>extraUpdates</code>:
: <math>\text{Invinciblity frame}=\left\lceil\frac{\textit{localNPCHitCooldown}}{\textit{extraUpdates}+1}\right\rceil</math>
: Directly displays <code>-1</code> if <code>localNPCHitCooldown = -1</code>, since some of these have non-default <code>extraUpdates</code> value.
* For global: displays <code>10</code> if no length parameter is presented.

; penetrate or p
The <code>penetrate</code> stat of the projectile. Displays {{tl|infinity}} if the value is <code>-1</code>.

; note or second unnamed parameter
Write notes here.
